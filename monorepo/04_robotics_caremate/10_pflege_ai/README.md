# 10 — ИИ в Pflege

**Тема каталога:** #26
**Приоритет:** Б (10)
**Уровень готовности:** ★★★★

## Что это

Применение ИИ в немецкой системе ухода (Pflege): помощь медперсоналу, оптимизация смен, оценка нагрузки, поддержка решений.

## Применение в Pflegeheim

| Задача | Как помогает ИИ |
|--------|----------------|
| Документация ухода | Автоматическая запись из голоса/видео |
| Планирование смен | Оптимизация под нагрузку и квалификацию |
| MDK-проверки | Подготовка отчётов автоматически |
| Pflegegrad-оценка | Помощь медперсоналу в правильной оценке |
| Контроль приёма лекарств | Распознавание лиц + контроль времени |
| Связь с семьёй | Автоматические сводки родственникам |
| Мотивация подопечных | Персональные диалоговые ИИ-агенты |

## Контекст в Германии

- **4.1 млн Pflegebedürftige** в Германии
- **Дефицит 500 000 медперсонала** к 2030
- **Pflegeversicherung тратит €50+ млрд/год**
- **Аналогов ИИ-инструментов мало** на немецком рынке

## Регуляторика

| Стандарт | Что покрывает |
|----------|---------------|
| **SGB XI** | Pflegegrad 1-5, основные правила |
| **DiGA-Verordnung** | Цифровые медицинские приложения (€20-40K сертификация) |
| **MDR** (Medical Device Regulation) | Если позиционируется как медустройство |
| **GDPR** | Персональные / медицинские данные |
| **AI Act EU** (2024) | Регулирование ИИ |

## Партнёры

- **AOK / Barmer / TK** (Krankenkassen) — могут субсидировать
- **Caritas / Diakonie / AWO / DRK** — операторы
- **MD (Medizinischer Dienst)** — экспертиза
- **Pflegekasse** — финансирование
