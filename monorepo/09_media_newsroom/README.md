# 09 — AI-Newsroom и медиа

**~10 разговоров · 4/5 готовности**
**Главный источник:** разговор #604 — AI-цепочка для новостей

## О чём этот кластер

Кластер собирает все материалы по **автоматизации журналистики, OSINT, медиа-конвейерам**. Главный актив — **полная AI-цепочка** для генерации новостей: мониторинг → сбор → NER → векторное хранение → RAG → генерация → публикация.

## Зачем это нужно

1. **AI-newsroom** — самостоятельный продукт для редакций
2. **Контент-генерация для других проектов** — SozialPlaner, CareMate, НейроПортал
3. **OSINT-сервисы** — для журналистов, исследователей, юристов
4. **SkyMediaHub-интеграция** — съёмка дронов → автоновости
