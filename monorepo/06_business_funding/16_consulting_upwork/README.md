# 16 — Upwork-консалтинг по SGB

**Цена:** €50-150 за документ
**Старт:** сразу
**Прогноз:** €250-1500/мес.

## Что это

Услуга на **Upwork / Fiverr / Malt**: «Написание Widerspruch / Antrag / Klage по SGB IX/XII».

Самый быстрый путь к **первым деньгам**: профиль создаётся за день, заказы могут пойти через неделю.

## Услуги

| Услуга | Цена | Срок выполнения |
|--------|------|-----------------|
| Widerspruch (1 страница) | €50-80 | 1-2 дня |
| Stellungnahme | €60-90 | 1-2 дня |
| Antrag PB полный пакет | €120-180 | 3-5 дней |
| Klage в Sozialgericht | €150-300 | 5-7 дней |
| Master Dossier (полное оформление дела) | €200-400 | 7-14 дней |
| Консультация (1 ч) | €60-100 | по записи |

## Профиль

**Заголовок:** «German Social Law Specialist (SGB IX/XII) — Widerspruch, Antrag, Klage»

**Описание:**
- 5+ лет опыта с SGB IX/XII (личный + помощь другим)
- 50+ успешных Widerspruch
- Знание DE/RU/UA/EN
- Ведение дел до Bundessozialgericht

**Услуги:**
- Перечень с ценами
- Примеры (анонимизированные)

**Сертификаты/портфолио:**
- Если будут публикации (статьи, книга)

## Маркетинг

1. **Upwork-bidding** — на новые заказы (первые 10-20)
2. **LinkedIn outreach** — российская диаспора в DE
3. **Reddit r/germany_russian** — ответы в комментариях
4. **Facebook-группы** — экспертные ответы
5. **Своя страница** на kanzlei.de (если возможно без юр. лиц.)

## Этический момент

Автор **не адвокат** в юр. смысле. Услуги позиционируются как:
- **Содержательная подготовка документов**, а не «представительство в суде»
- **Совет по навигации** в системе, а не «правовое консультирование»

Если сложный кейс — **переадресация к адвокату** (партнёрство).
