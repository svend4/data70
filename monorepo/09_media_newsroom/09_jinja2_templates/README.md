# 09 — Jinja2-шаблоны генерации

**Уровень готовности:** ★★★★

## Что это

**Jinja2** — шаблонизатор Python, идеально подходит для **гарантированной структуры** генерируемого контента в AI-newsroom.

Идея: LLM заполняет переменные, не пишет всю статью свободно. Это даёт:
- Стабильную структуру
- Контроль качества
- Меньше галлюцинаций

## Пример шаблона новости (concept)

```jinja2
# {{ title }}

**{{ date }}** — {{ location }}

{{ lead_paragraph }}

## Что произошло

{{ description }}

{% if quotes %}
## Комментарии

{% for quote in quotes %}
> «{{ quote.text }}»
> 
> — {{ quote.author }}, {{ quote.role }}
{% endfor %}
{% endif %}

## Что дальше

{{ next_steps }}

---

*Источники: {{ sources | join(", ") }}*
*Опубликовано: {{ publication_date }}*
*Автор: AI-Newsroom*
```

LLM получает структуру и заполняет каждую переменную:
- `title` — заголовок
- `lead_paragraph` — лид
- `description` — основной текст
- `quotes` — массив цитат
- и т.д.

## Преимущества

| Плюс | Описание |
|------|----------|
| **Структура гарантирована** | Журнал получает статью в правильном формате |
| **Меньше галлюцинаций** | LLM ограничен полями |
| **Версионируемость** | Шаблоны в Git |
| **A/B-тестирование** | Разные шаблоны → разные стили |
| **Перевод** | Один шаблон, переменные на разных языках |

## Шаблоны для разных типов

- Краткая новость (200 слов)
- Статья (500-1000)
- Расследование (2000+)
- Дайджест (5-10 коротких)
- Аналитика (с цифрами и графиками)
- Интервью (Q&A формат)
