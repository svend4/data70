# 07 — Водный дрон на подводных крыльях

**Категория:** изобретение (см. `07_inventions_patents/05_water_drone`)
**Уровень готовности:** ★★★

## Что это

**Водный БПЛА дельтавидной формы** на гидрофойлах для морского применения. Подводные крылья поднимают корпус над водой при движении на скорости — снижение сопротивления, экономия энергии.

## Принцип

```
Стоит:                   Движется:
═════════════ (вода)     ═══════ (вода)
░░░░░░░░░░               
██████████ корпус         ▼ ▼ ▼ гидрофойлы
                          ──────── корпус (над водой)
```

При V < 8-12 км/ч — обычное плавание.
При V > 12 км/ч — подъём над водой на гидрофойлах.

## Применения

| Сценарий | Полезная нагрузка |
|----------|-------------------|
| Морской мониторинг | Камеры, гидрофоны |
| Поисково-спасательные операции | Сонары, тепловизоры |
| Биомониторинг (китообразные, кораллы) | Подводные камеры |
| Военная разведка | Радиоэлектроника |
| Доставка между судами | Малые грузы |

## Спецификация (концепт)

| Параметр | Значение |
|----------|----------|
| Длина | 1-2 м |
| Масса | 5-15 кг |
| Скорость крейсерская | 30-60 км/ч |
| Время автономности | 4-8 ч (батареи) или дни (гибрид с солнечной) |
| Глубина гидрофойлов | 0.3-0.5 м |
| Управление | Дистанционное / автономное (по GPS-маршруту) |
