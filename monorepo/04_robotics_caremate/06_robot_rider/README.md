# 06 — Гибридный робот-всадник

**Категория:** изобретение (см. `07_inventions_patents/08_robot_rider`)
**Уровень готовности:** ★★★

## Что это

**Двухмодульный робот**: нижний модуль на колёсах/ногах + верхний с манипуляторами. Аналогия: «робот-всадник» сидит на «роботе-собаке».

## Принцип

```
   ╔═════════╗     ← верхний модуль (манипуляторы, ИИ)
   ║  TORSO  ║      • Плечи + 2 руки
   ║         ║      • Голова с камерами
   ║  ▼  ▼   ║      • Бортовой компьютер
   ╚═══╧═╧═══╝
       │ │ ← разъём (универсальный)
   ╔═══╪═╪═══╝
   ║ █ █ █ █ ║    ← нижний модуль (мобильность)
   ║         ║      • Может быть: 4-ногий, гусеничный, колёсный
   ╚═════════╝      • На воде, на снегу, на лестницах
```

## Преимущества

1. **Модульность** — один торс на разных «носителях»
2. **Адаптация к среде** — выбор нижнего модуля по местности
3. **Ремонтопригодность** — заменить только сломанный модуль
4. **Цена** — несколько носителей дешевле, чем несколько целых роботов
5. **Эволюция** — улучшение торса не требует замены носителя

## Применения

| Сценарий | Нижний модуль | Верхний модуль |
|----------|---------------|----------------|
| Логистика на складе | Колёсный | Гриппер |
| Спасение в катастрофах | 4-ногий | Камера + рука |
| Промышленность | Гусеничный | Сварочный аппарат |
| Уход за пожилыми | Колёсный (тихий) | Мягкие манипуляторы |
| Хирургия | Стационарный | Прецизионные руки |

## Аналоги

- **Boston Dynamics Atlas** — цельный, не модульный
- **Boston Dynamics Spot + Arm** — близко к идее (Spot + рука сверху)
- **Tesla Optimus** — цельный
- **Авторская концепция** — полностью модульная, открытый стандарт разъёма

## Стандартизация интерфейса

Ключ к успеху — **открытый стандарт разъёма** между модулями. Аналогия: USB-C для мобильного, но для роботов.

Параметры разъёма:
- Механическое крепление
- Электропитание (24-48 В)
- Высокоскоростная связь (Ethernet/USB-C)
- Система охлаждения (опционально)
