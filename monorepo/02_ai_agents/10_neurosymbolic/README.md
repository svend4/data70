# 10 — Neurosymbolic AI

**Тема каталога:** #9
**Кластер A-J:** A
**Уровень готовности:** ★★★

## Что это

**Neurosymbolic AI** — гибрид нейронных сетей (распознавание паттернов) и символьной логики (правила, рассуждения). Цель — соединить «интуицию» нейросети с «дисциплиной» логики.

Эталонные направления: differentiable theorem proving, neuro-symbolic theorem proving, AlphaProof (DeepMind).

## Зачем это нужно

1. **Юридические рассуждения** — закон требует символьной логики
2. **Объяснимость** — символьная часть даёт цепочку рассуждений
3. **Надёжность** — правила гарантируют отсутствие галлюцинаций в критичных местах
4. **Интеграция знаний** — онтологии, графы знаний

## Применение в проектах автора

### SozialPlaner
- Нейросеть распознаёт ситуацию пользователя
- Символьный движок применяет дерево правил SGB IX
- Результат: однозначный ответ с цепочкой обоснования

### CareMate
- Нейросеть анализирует биометрию
- Правила: «если ЧСС >120 в течение 5 минут → тревога»
- Прозрачность для медперсонала

### Newsroom
- Нейросеть извлекает факты
- Логика проверяет противоречия
- Фактчекинг как нейросимвольная задача
