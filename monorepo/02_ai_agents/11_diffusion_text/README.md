# 11 — Диффузионная генерация текста

**Тема каталога:** #10
**Кластер A-J:** A
**Уровень готовности:** ★★

## Что это

**Diffusion language models** — генерация текста как обратный процесс зашумления (как в Stable Diffusion для картинок). Эталоны: SEDD, Score Entropy Discrete Diffusion (Stanford), MD4, Mercury Coder (Inception Labs, 2025).

В отличие от авторегрессионных LLM (генерация слева направо), диффузионные модели генерируют **параллельно** и могут редактировать любой фрагмент.

## Преимущества

1. **Скорость** — параллельная генерация всего ответа за <1 сек
2. **Редактирование** — можно «зафиксировать» часть текста и менять остальное
3. **Контроль** — лучше следуют структурным ограничениям
4. **Связь с #4 (LCM)** — естественная иерархия концептов

## Применение в проектах автора

### Юридические шаблоны
- Зафиксированы реквизиты сторон, параграфы
- Диффузия дополняет аргументы и факты
- Результат: всегда формально корректный документ

### Новости (Newsroom)
- Структура (заголовок-лид-основная часть-цитата-вывод) фиксирована
- Диффузия «дорисовывает» содержание

### Книга по SGB
- Каркас (24 пункта мегашаблона) фиксирован
- Диффузия заполняет каждый пункт по конкретной услуге
