# 06 — Sparse Expert Models / Mixture of Experts (MoE)

**Тема каталога:** #2
**Кластер A-J:** A
**Приоритет:** Б (9)
**Уровень готовности:** ★★★

## Что это

**Mixture of Experts (MoE)** — архитектура нейросетей, в которой сеть разделена на множество «экспертов», и для каждого входа активируется лишь подмножество (sparse activation).

Эталонные представители: Mixtral 8x7B (Mistral), DeepSeek-V3, GPT-4 (предположительно), Switch Transformer (Google).

## Как это работает

```
Вход → Router → выбирает 2 из 8 экспертов
                              ↓
                        Эксперт 1 ┐
                                  ├→ агрегация → выход
                        Эксперт 5 ┘
```

Параметров — много (например, 56B). Активных за один проход — мало (например, 14B).

## Зачем это нужно автору

1. **Производительность** — большие модели работают быстро
2. **Специализация** — эксперты могут быть «правовым», «медицинским», «техническим»
3. **Аналогия с темой #14 (биомодель)** — эксперты как «пчёлы» в пчелином слое

## Конкретные применения

- **SozialPlaner** — MoE с экспертом по SGB IX, экспертом по SGB XII, и т.д.
- **CareMate** — эксперт по Pflege, эксперт по медицине, эксперт по бюджету
- **Newsroom** — эксперт по политике, эксперт по экономике, эксперт по технике
