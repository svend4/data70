# 07 — Large Cognitive Models (LCM)

**Тема каталога:** #4
**Кластер A-J:** A
**Приоритет:** В (8 — долгосрочно)
**Уровень готовности:** ★★

## Что это

**Large Cognitive Models** — следующая ступень эволюции от LLM. Модели, которые работают **не с токенами, а с концептами**: единица мышления — не слово, а смысловой блок (предложение, идея, концепция).

Идея Meta (2024): «Large Concept Models» — генерация на уровне предложений, а не слов. Языково-независимые.

## Чем отличается от LLM

| Признак | LLM | LCM |
|---------|-----|-----|
| Единица | Токен (slово/часть слова) | Концепт (предложение/идея) |
| Контекст | 8K-1M токенов | Существенно больше идей |
| Язык | Языково-зависимая | Языково-независимая |
| Вычисления | Атомарно | Иерархично |
| Аналог в природе | Звуки/буквы | Мысли |

## Почему это важно автору

1. **Многоязычные продукты** — естественная работа DE/RU/EN/UA без потери смысла
2. **Глубокий анализ** — RAG над «концептами», а не над фрагментами текста
3. **Эффективность** — меньше токенов в контексте = быстрее и дешевле
4. **Связь с темой #10** (диффузионная генерация) — нелинейная генерация на уровне идей

## Применение

- **НейроПортал** — LCM может анализировать тысячи документов как «концепты», а не как «токены»
- **Книга по SGB** — автоматический перевод DE↔RU без потери юридического смысла
- **CareMate** — диалог на уровне «намерения», а не «команды»
