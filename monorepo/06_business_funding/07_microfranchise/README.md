# 07 — Микрофраншизы

**Тема каталога:** #70 (хаб 12+ связей)
**Приоритет:** А (11)
**Уровень готовности:** ★★★★

## Что это

**Микрофраншиза** — малая бизнес-модель «под ключ»: пакет из готовых n8n-сценариев + бренда + ПО + инструкции, который продаётся за €5-50. Покупатель получает **готовый микро-бизнес**.

Авторская концепция объединяет:
- Блокчейн (учёт продаж и роялти)
- ИИ (для масштабируемости)
- Автоматизацию (готовые сценарии)

## Примеры микрофраншиз

| Микрофраншиза | Что входит | Цена |
|---------------|-----------|------|
| **Дропшипинг AliExpress** | Сценарий n8n + список товаров + landing | €19 |
| **PDF-summarizer SaaS** | n8n + PDFGenie + Stripe | €39 |
| **Subscription box на Shopify** | Готовый магазин + автоматизация | €99 |
| **Pflege-Bot для Pflegedienste** | Bot + контент SGB | €299 |
| **Sozialhelfer-Service** | Telegram + база шаблонов | €199 |

## Бизнес-модель «маркетплейс микрофраншиз»

```
Автор (создаёт микрофраншизу)
   ↓
Платформа (хостит + промотирует) — 15-20% комиссия
   ↓
Покупатель (запускает свой микро-бизнес)
   ↓
Возможные роялти автору (через смарт-контракт)
```

## Зачем это нужно

1. **Демократизация предпринимательства** — каждый может купить готовый бизнес за €19-99
2. **Ниши, которые невыгодны крупным** — узко-специализированные
3. **Бесконечная композиция** — сочетание микрофраншиз для большего бизнеса
4. **Связь с темой #70** — хаб с 12+ связями
