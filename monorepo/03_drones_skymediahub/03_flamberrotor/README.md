# 03 — FlamberRotor: волнообразные лопасти пропеллера

**Категория:** изобретение (см. `07_inventions_patents/02_flamberrotor`)
**Уровень готовности:** ★★★

## Что это

**FlamberRotor** — лопасти вертолётного/дронного пропеллера с **волнообразным профилем** по аналогии с мечом-фламбергом (волнистый клинок).

Идея: волнообразная передняя кромка лопасти меняет вихревые эффекты, что потенциально:
- Снижает аэродинамический шум
- Стабилизирует подъёмную силу при определённых режимах
- Уменьшает срыв потока

## Природный прототип

Передняя кромка плавника горбатого кита имеет волнообразные «бугорки» (tubercles) — это снижает срыв потока на больших углах атаки. На этом основан FlamberRotor.

## Аналоги в индустрии

| Решение | Принцип | Производитель |
|---------|---------|--------------|
| **Tubercle technology** | Бугорки, как у кита | WhalePower (CA) |
| **Зубчатая кромка** | Серрейторы | NASA, Airbus |
| **Изогнутые лопасти** | Curved blades | Sikorsky |
| **FlamberRotor** | Полная волна | Авторская концепция |

Особенность авторской концепции — **полная волна** (фламбергская), а не локальные бугорки.

## Что нужно для подтверждения

1. **CFD-моделирование** в OpenFOAM (open-source)
2. **3D-печать прототипа** лопасти
3. **Замеры в аэродинамической трубе** (например, в Fraunhofer)
4. **Сравнение** с обычной лопастью по: тяга, шум, КПД

## Применение

- **Тихие гражданские дроны** для городской среды (Lieferdrohne, Inspektion)
- **Длительные полёты** с экономией энергии
- **Военное применение** — снижение шумовой сигнатуры
