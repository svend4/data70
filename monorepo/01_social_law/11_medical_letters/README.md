# 11 — Медицинские письма для соцведомств

**Кластер:** 01_social_law
**Уровень готовности:** ★★★★★
**Источник:** разговор #1 — 1365 сообщений, 2.6 МБ (один из самых проработанных)

## Что это

Серия шаблонов и реальных писем для общения с **врачами и медицинскими экспертами** в контексте социальных дел. Эти письма — не диагноз, а **юридический инструмент**: они должны быть составлены так, чтобы Sozialamt, Pflegekasse, Versorgungsamt не могли их отклонить.

## 6 типов медицинских справок

1. **Bescheinigung über Pflegebedarf** — подтверждение нужды в уходе
2. **Bescheinigung über GdB** — подтверждение степени инвалидности
3. **Stellungnahme zum MDK-Gutachten** — позиция по экспертизе MDK
4. **Bescheinigung über Hilfsmittelbedarf** — нужда в средствах реабилитации
5. **Bescheinigung über Mobilitätseinschränkung** — ограничения мобильности
6. **Bescheinigung über Pflegebedürftigkeit nach SGB XI** — для Pflegegrad

## Что отличает «правильное» письмо

| Признак | Что это значит |
|---------|----------------|
| Формальный титул | «Ärztliche Bescheinigung» / «Stellungnahme» |
| Заголовок-цель | «Zur Vorlage beim Sozialamt» |
| Ссылка на параграфы | «im Sinne des § 14 SGB XI» |
| Конкретика | «kann sich nicht selbständig waschen», а не «hat Probleme» |
| Количественная оценка | «benötigt 4 Stunden Hilfe täglich» |
| Подпись и Stempel врача | без штампа письмо могут отклонить |

## Сценарии использования

- Перед подачей Antrag PB — пакет из 3-5 врачебных Bescheinigung
- Для Widerspruch против отказа в Pflegegrad — Stellungnahme к MDK-Gutachten
- Для Klage в Sozialgericht — комплект медицинских доказательств

## Что собрано

- Шаблоны 6 типов справок
- Список «что нужно сказать врачу», чтобы он написал правильно
- Список запрещённых формулировок («kann manchmal»)
- Список ключевых формулировок («ist auf Hilfe angewiesen»)
- Реальная переписка с MDK и реакции
