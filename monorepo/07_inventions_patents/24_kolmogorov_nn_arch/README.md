## Суть

**KAN (Kolmogorov-Arnold Networks)** — альтернативная архитектура нейронной сети, в которой:
- Веса находятся **на рёбрах**, не на узлах
- Каждый вес — обучаемая функция (B-сплайн), а не скаляр
- Структура основана на теореме Колмогорова-Арнольда

## Аналогия с MLP

| Признак | MLP | KAN |
|---------|-----|-----|
| Веса | На узлах | На рёбрах |
| Активация | Фиксированная | Обучаемая |
| Интерпретируемость | Низкая | Высокая |
| Размер | Большой | Может быть меньше |

## Применение в проектах автора

1. **Юридический ИИ** — интерпретируемость критична для SozialPlaner
2. **Медицинский ИИ** (CareMate) — прозрачность для врачей
3. **Edge-ИИ** — KAN могут быть компактнее MLP
4. **Forth-интеллект** — потенциальный target

## Защита

KAN — **академическая концепция** (MIT 2024), уже опубликована. Защищается **авторами** (Liu et al.).

- **Конкретные применения** KAN к юридическим задачам (если новизна)
