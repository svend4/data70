# 12 — UFO-дрон-щит

**Уровень готовности:** ★★

## Что это

Концепция: купольный (тарелкообразный) дрон с защитной оболочкой, который может **выступать в роли «щита»** — закрывать собой объекты от дождя, наблюдения сверху, осколков.

## Принцип

```
   ┌───────────┐  верхняя оболочка (купол, защита)
   │   ─ ─ ─   │  кольцевой ротор
   │   корпус  │  внутренняя капсула с электроникой
   │   ─ ─ ─   │  кольцевой ротор
   └───────────┘
```

Кольцевые роторы внутри купола — низкий шум, защита от внешнего воздействия.

## Применения

- **Защита от дождя/снега** для уличных мероприятий, киносъёмки
- **Мобильное укрытие** для гумбанов
- **Защита от наблюдения сверху** — приватность на улице
- **Художественные инсталляции**
