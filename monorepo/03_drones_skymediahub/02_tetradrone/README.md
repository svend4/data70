# 02 — TetraDrone / TetraFly

**Тема каталога:** #63
**Категория:** изобретение (см. `07_inventions_patents/01_tetradrone`)
**Уровень готовности:** ★★★★

## Что это

**TetraDrone** — корпус дрона, производимый на стандартной упаковочной линии Tetra Pak. Многослойная структура (картон + полимер + алюминий) даёт:
- Лёгкость
- Влагостойкость
- Биоразлагаемость основы
- **Стоимость рамы <€5**

## Принцип

```
Лист Tetra Pak (А4, плотность 350 г/м²)
   ↓ лазерная резка / штамп
Развёртка корпуса (плоская)
   ↓ сгиб + клей / термосварка
Готовый корпус дрона (250-450 г взлётного веса)
```

## Сравнение с Sypaq PPDS (Австралия)

| Признак | Sypaq PPDS | TetraDrone (авторская концепция) |
|---------|-----------|----------------------------------|
| Материал | Картон | Tetra Pak |
| Влагостойкость | Низкая | Высокая |
| Стоимость рамы | ~€5 | <€5 |
| Биоразлагаемость | Высокая | Высокая (с оговорками на полимер) |
| Серийное производство | Своя линия | **Готовая линия Tetra Pak** |
| EU присутствие | Нет | Да (заводы по всей Европе) |

## Ключевая инновация

**Использование готовой производственной инфраструктуры Tetra Pak** в Европе:
- Тысячи упаковочных линий уже работают
- Адаптация одной линии = переоборудование за дни, не годы
- Поддержка местного производства (важно для EU стратегии независимости)
