- Четырёхуровневая пирамида знаний (хаб 12+ связей)
- Инфо-Бонсай — личная база знаний
- YAML-инфоконвейер
- ВикиИИпедия
- LMS + SLM
- TiddlyWiki
- Трёхуровневая модель управления
- Коллективные профили ИИ
- Виртуальная нейронация
- Археология идей
- KBLaM как методология
