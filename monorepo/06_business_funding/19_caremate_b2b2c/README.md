# 19 — CareMate B2B2C SaaS + Hardware Bundle

**Тип:** B2B2C SaaS + Hardware Bundle
**MVP бюджет:** €150-220K
**Срок MVP:** 5-6 мес.
**Источник:** `part14_business_models.md` раздел 14.3

## Что это

Бизнес-модель для **CareMate Total** — суперпроекта для рынка ухода в Германии.

## Тарифы

| Тариф | Цена | Аудитория |
|-------|------|-----------|
| **Family** | €14.90/мес | Семья с 1 пожилым |
| **Family Plus** | €29.90/мес | + IoT-датчики (аренда), голос ИИ, 20+ сценариев |
| **Care Facility** | €4.90/мес/койка | Pflegeheim |
| **Enterprise** | индивидуально | Krankenkassen, B2B-API |

## Дополнительные источники

- **IoT-комплект** (продажа/аренда) — €99-299
- **Партнёрства со страховыми** — Krankenkasse субсидирует €50-100/мес
- **White-label** для Caritas / Diakonie / AWO / DRK
- **Анонимизированные данные** — для фарм/исследователей

## Рынок (Германия)

- **4.1 млн Pflegebedürftige**
- **Pflegeversicherung €50+ млрд/год**
- **Дефицит 500K медперсонала** к 2030
- **TAM Германия:** €2-5 млрд/год

## Путь к окупаемости

```
Мес 1-6:    MVP, пилот 3 Pflegeheim (50 коек)
Мес 7-12:   10 учр. + 100 семей = €8K MRR
Мес 13-18:  Партнёрство с 1 Krankenkasse, DiGA-сертификация
Мес 19-24:  50 учр. + 500 семей = €40K MRR
Мес 24:     Безубыточность
Мес 30-36:  Австрия, Швейцария, Нидерланды
```

- Кол-во подключённых учреждений
