## Что это

**LiveMCP-101** — бенчмарк для оценки качества LLM-агентов, использующих MCP-инструменты. Тестирует в живых задачах: поиск, парсинг, генерация, проверка. 101 задача с эталонными решениями.

## Зачем нужен

1. **Качество агентов** — без бенчмарка нельзя сравнить
2. **Оптимизация** — какие модели лучше с какими MCP-серверами
3. **Маркетинг** — публикация результатов как доказательство компетенции
4. **Pre-seed deck** — «наши агенты делают 87% LiveMCP-101»

## Применение в проектах автора

- **SozialPlaner** — собственные MCP-серверы (sgb-rag, widerspruch-gen) проходят LiveMCP-101
- **НейроОС** — встроенный «test mode» для микроагентов
- **Pre-seed deck** — графики результатов

## Что измеряет

| Метрика | Что |
|---------|-----|
| Pass rate | % правильных ответов |
| Tool use efficiency | Сколько вызовов в среднем |
| Latency | Время на задачу |
| Cost | API-стоимость |
| Robustness | Устойчивость к шуму |
