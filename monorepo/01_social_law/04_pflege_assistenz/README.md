# 04 — Pflegeassistenz: расчёт, часы, ставки

**Кластер:** 01_social_law
**Уровень готовности:** ★★★★★

## Что это

Pflegeassistenz — ассистенция по уходу. Финансируется из Pflegekasse (SGB XI) и/или из Persönliches Budget (SGB IX). Расчёт часов и ставок — главный спорный пункт.

## 4-уровневая детализация

| Уровень | Содержание |
|---------|-----------|
| 1. Описание | Что именно делает ассистент: гигиена, питание, мобильность, сопровождение |
| 2. Часы | Сколько часов в день/месяц по каждому виду помощи |
| 3. Ставки | €/час по виду услуги (дневной, ночной, выходной) |
| 4. Суммы и обоснование | Итоговая месячная сумма + ссылки на нормативы |

## Типичные ставки (Sachsen, 2025)

| Вид помощи | Ставка €/час |
|-----------|-------------|
| Базовая ассистенция (день) | 22-28 |
| Базовая ассистенция (ночь, надбавка) | +30-50% |
| Квалифицированная ассистенция | 35-45 |
| Перевозка и сопровождение | 18-22 |
| Выходной/праздник (надбавка) | +25-50% |

## Ключевые параграфы

- **§ 78 SGB IX** — Assistenzleistungen для людей с ограничениями
- **§ 64b SGB XII** — Hilfe zur Pflege
- **SGB XI** — Pflegegrad 1-5, Geldleistungen
- **§ 84 SGB IX** — Doppelausstattung (двойное обеспечение)

## Конфликтные точки

1. **Двойной учёт** SGB XI + SGB IX → ведомства часто отказывают
2. **Произвольные коэффициенты** — Sozialamt сокращает часы без обоснования
3. **Ночные часы** — занижаются или отрицаются
4. **Сопровождение к врачу** — переводится на Krankenkasse, та отказывает
