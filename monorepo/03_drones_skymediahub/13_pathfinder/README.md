# 13 — Pathfinder: разведывательный дрон

**Уровень готовности:** ★★

## Что это

Малый разведывательный дрон, фокус на:
- Дальний полёт (1-3 ч)
- Малая заметность (тихие лопасти, например `03_flamberrotor`)
- Автономность (полёт по маршруту без оператора)
- Передача данных в реальном времени

## Спецификация (концепт)

| Параметр | Значение |
|----------|----------|
| Тип | Гибрид крыло+мультиротор |
| Размах | 1.2 м |
| Взлётный вес | 1.5-2 кг |
| Время полёта | 90-180 мин |
| Дальность | 100-200 км |
| ИИ | Бортовой Forth-интеллект на ESP32 |

## Применения

- Гражданские: картография, мониторинг лесов, поиск пропавших
- Dual-use: периметр-разведка для военных
