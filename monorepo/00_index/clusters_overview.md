## Кластеры из аналитики (A-J) и как они отражаются

Аналитический документ `part12_catalog.md` использует 10 кластеров (A-J). Эта папочная структура переупаковывает их в 11 «продуктовых» кластеров. Соответствие:

| Кластер A-J | Папка-кластер монорепозитория |
|-------------|-------------------------------|
| A — Архитектура ИИ нового поколения | 02_ai_agents (часть 03–14) |
| B — Инфраструктура и платформы ИИ | 02_ai_agents (часть 15–25) |
| C — Персональная база знаний | 08_knowledge_methodology + 05_software_automation |
| D — ИИ для обычных людей | 04_robotics_caremate + 10_health_accessibility |
| E — Виртуальный офис и бизнес | 05_software_automation + 06_business_funding |
| F — Интернет 3.0 | 05_software_automation/12, 13 + 08_knowledge_methodology/12, 13 |
| G — Дроны и робототехника | 03_drones_skymediahub + 04_robotics_caremate |
| H — Языки и формализмы | 07_inventions_patents (09, 17–24) |
| I — Блокчейн и финансы | 06_business_funding (07, 09, 11) |
| J — Цифровые кочевники | 06_business_funding/10, 11 + 10_health_accessibility/10 |

## Кластеры с наибольшим коммерческим потенциалом

1. `01_social_law` — готовый рынок 13 млн людей с GdB в Германии, низкий бюджет MVP
2. `04_robotics_caremate` (CareMate) — B2B рынок 15 000 учреждений
3. `02_ai_agents` (НейроОС) — глобальный SaaS, низкая стоимость входа
4. `05_software_automation` (Internet Function OS) — реальная боль рынка no-code
5. `03_drones_skymediahub` — гранты dual-use, agro
