## Связи между подпапками

- `01_pyramid_4_levels` — базовая концепция, на которой строится `02_info_bonsai`
- `03_yaml_pipelines` — реализация перетекания между уровнями пирамиды
- `04_archeology_of_ideas` — частный случай YAML-пайплайна
- `09_kblam_methodology` — альтернатива RAG
- `13_neuro_nation` — логическое расширение `12_collective_ai_profiles`

## Связь с `02_ai_agents`

| Методология | Технология |
|-------------|-----------|
| Пирамида знаний | RAG-движок + структура |
| Инфо-Бонсай | ChromaDB + SQLite + LangChain |
| YAML-пайплайн | Airflow / Prefect |
| KBLaM | LLM с встроенной БЗ |
| LMS + SLM | Phi-2/Phi-3.5 + Moodle |

## Связь с `05_software_automation`

- Internet Function OS архитектурно отражает пирамиду знаний
- Composer = инструмент для пирамиды
- Blueprints = практический уровень
