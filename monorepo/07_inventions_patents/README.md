# 07 — Изобретения и патентабельные идеи

**24 концепции с признаками новизны и применимости.**
**Источник:** `analysis_03_inventions.md` + `00_index/inventions_index.md`

## О чём этот кластер

Кластер собирает все идеи, обладающие потенциальной патентной новизной и промышленной применимостью. Каждая карточка содержит:
- Описание сути
- Сравнение с существующими аналогами (prior art)
- Оценку новизны и применимости
- Рекомендации по подаче патентной заявки
