## Что это

**Exo** (exo-explore/exo, GitHub) — open-source проект, позволяющий **распределить запуск LLM по нескольким бытовым устройствам**: ноутбуки, телефоны, Raspberry Pi. Превращает домашнюю сеть в мини-кластер.

## Принцип работы

```
Llama 3.1 70B (140 ГБ FP16, 35 ГБ Q4)
   ↓ разделение по слоям
Устройство 1 (ноутбук Mac M2, 32 ГБ) — слои 1-30
Устройство 2 (старый Mac, 16 ГБ)     — слои 31-50
Устройство 3 (iPhone 15)              — слои 51-80
   ↓
Объединённый ответ через mesh
```

## Зачем это нужно автору

1. **Дешевизна** — собрать «70B-модель» из старого железа за €0
2. **Приватность** — данные никогда не выходят из локальной сети
3. **Масштабируемость** — добавить устройство = увеличить пул памяти
4. **Связь с темой #14 (биомодель)** — рой устройств как пчелиный слой

## Конкретное применение

- **НейроОС** — пользователь добавляет устройства как «пчёл» → пул вычислений растёт
- **CareMate в Pflegeheim** — старые планшеты медперсонала как inferring-сеть
- **Цифровой Караван** — вычислительный пул кочевников
- **SozialPlaner offline** — запуск 70B-модели без облака

## Технологии

- **libp2p** — P2P-сеть
- **Tinygrad** (по выбору) — фреймворк
- **Подсетевая балансировка** — динамическое распределение слоёв
