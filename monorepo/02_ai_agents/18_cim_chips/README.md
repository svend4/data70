## Что это

**Compute-in-Memory (CiM)** — парадигма, в которой вычисления выполняются прямо в ячейках памяти, без переноса данных на CPU/GPU. Решает «фон-неймановское бутылочное горлышко» — главный энергетический потолок современных чипов.

**Techifab** — компания, разрабатывающая мемристорные нейроморфные чипы.

## Почему это критично для будущего ИИ

1. **Энергоэффективность** — в 100-1000 раз меньше энергии на одну операцию
2. **Скорость** — нет передачи данных между памятью и АЛУ
3. **Edge-ИИ** — модели в наушниках, часах, сенсорах
4. **Связь с #3 (1-Bit LLM)** — бинарные сети идеально ложатся на CiM-чипы
