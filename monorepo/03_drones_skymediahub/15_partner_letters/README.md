# 15 — Письма-запросы партнёрам

**Уровень готовности:** ★★★★★ (готовы к отправке)

## Что это

Реальные письма на немецком, **готовые к отправке**, обращённые к ключевым технологическим и финансовым партнёрам в DACH. Архив содержит готовые тексты + контактную информацию.

## Адресаты

### 1. Fraunhofer IPMS Dresden
- **Тема:** партнёрство по оптике 4K-камеры (`08_4k_compound_camera`) и FlamberRotor (`03_flamberrotor`)
- **Запрос:** доступ к аэродинамической трубе и оптическому тестированию
- **Контакт:** ipms@ipms.fraunhofer.de
- **Статус:** письмо готово, не отправлено

### 2. TechHUB SVI Sachsen
- **Тема:** соинвестирование MVP TetraDrone (`02_tetradrone`)
- **Запрос:** инкубация + GründerGeld Sachsen
- **Контакт:** через Smart Systems Hub Dresden
- **Статус:** письмо готово

### 3. Bundeswehr-связанные структуры
- **Тема:** концепция конверсии автопромышленности под UGV (`17_bundeswehr_conversion`)
- **Запрос:** аудиторская встреча для оценки концепции
- **Контакт:** через Cyber Innovation Hub (CIH)
- **Статус:** концепт готов, формат подачи требует уточнения

### 4. Bayern Innovativ
- **Тема:** грант €200K на SkyMediaHub Bavaria
- **Запрос:** заявка на конкурс
- **Контакт:** через grant@bayern-innovativ.de
- **Статус:** заявка в работе

### 5. Wingcopter (DE) / Quantum Systems (DE)
- **Тема:** возможная кооперация по специальным аппаратам
- **Запрос:** информационная встреча
- **Статус:** письмо в подготовке

## Структура каждого письма

1. **Шапка** — отправитель/получатель
2. **Краткое представление** — кто такой автор, контекст
3. **Тема и контекст** — почему обращение именно к этому партнёру
4. **Конкретное предложение** — что именно предлагается
5. **Что нужно от партнёра** — конкретный запрос
6. **Что предлагается взамен** — IP, доли, кофинансирование
7. **Календарь** — предлагаемые даты встречи
8. **Контакты** — телефон, email, LinkedIn
