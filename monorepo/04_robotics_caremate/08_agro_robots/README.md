# 08 — Роботы в сельском хозяйстве

**Тема каталога:** #24
**Уровень готовности:** ★★★

## Что это

Сектор сельскохозяйственных роботов — UGV (наземные) и UAV (воздушные). В контексте автора:

- UAV: см. `03_drones_skymediahub/11_agro_scout`
- UGV: эта папка

## Категории сельхоз-роботов

| Категория | Задача | Примеры |
|-----------|--------|---------|
| **Тракторные UGV** | Пахота, посев, опрыскивание | John Deere autonomous |
| **Прополочные** | Удаление сорняков | EcoRobotix |
| **Сборочные** | Сбор урожая | Agrobot |
| **Мониторинговые** | Состояние растений, почвы | Saga Robotics |
| **Логистические** | Перевозка тары между полем и складом | Concept-роботы |
| **Авторская идея** | Рой малых TetraDrone-наземных | Связка `02_tetradrone` + UGV-преобразование |

## Связь с темой #57 (WILOS) и роем

Авторская идея: **рой малых сельхоз-UGV**, координируемых через WILOS, с TetraDrone-аэро-разведкой над ним. Такая комбинация даёт:
- Аэро-карта поля от Agro-Scout
- Наземная работа малых UGV (прополка, точечный полив)
- Координация через WILOS
