## Суть концепции

**Авторская методология управления информацией** с 4 уровнями:

```
СТРАТЕГИЧЕСКИЙ — общая картина, цели, принципы (редко обновляется)
       ↓ ↑
ТАКТИЧЕСКИЙ    — годовое/квартальное планирование
       ↓ ↑
ОПЕРАТИВНЫЙ    — недельные/ежедневные задачи
       ↓ ↑
ПРАКТИЧЕСКИЙ   — конкретные документы, переписка, заметки
```

## Алгоритмы движения

### Снизу вверх (хаос → структура → энциклопедия)
- Заметки → темы → разделы → знания
- IF/THEN-развилки в каждом узле
- Шаблоны имён файлов

### Сверху вниз (навигация → конкретика)
- Цель → подзадачи → действия → результаты
- Чек-листы на каждом уровне

## Чем отличается от других методов

| Метод | Уровней | Особенность |
|-------|---------|-------------|
| **Zettelkasten** | плоский | Никлас Луман, без иерархии |
| **PARA** (Tiago Forte) | 4 типа | Не иерархические уровни |
| **GTD** | по контекстам | Не иерархические |
| **OKR** | 2 уровня | Цель + ключевые результаты |
| **Авторская пирамида** | 4 уровня | Иерархическая + двунаправленные алгоритмы |
