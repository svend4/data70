# 03 — Маркетплейс n8n-сценариев

**Тема каталога:** #88
**Уровень готовности:** ★★★

## Что это

**Платформа продажи готовых n8n-автоматизаций** — каждый сценарий это «микрофраншиза» (см. `06_business_funding/07_microfranchise`). Бизнес-модель: 20% комиссия с каждой продажи.

## Категории сценариев

| Категория | Примеры |
|-----------|---------|
| **CRM-автоматизация** | Лиды из формы → CRM → email |
| **E-commerce** | Заказ → склад → доставка → отчёт |
| **Маркетинг** | Соцсети → анализ → дайджест |
| **HR** | Кандидат → ATS → собеседование |
| **Финансы** | Расходы → категоризация → отчёт |
| **Контент** | RSS → перевод → публикация |
| **AI-сценарии** | Документ → суммаризация → email |

## Технологии

- **Frontend:** Next.js + Stripe Checkout
- **Backend:** Supabase (auth, DB, storage)
- **Поиск:** Meilisearch
- **Оплата:** Stripe + PayPal

## Ценообразование

| Уровень | Цена | Тип |
|---------|------|-----|
| Free | €0 | Маркетинг-сценарии |
| Микро | €5-15 | Простые (1-3 шага) |
| Средние | €15-50 | Полноценные пайплайны |
| Корпоративные | €50-200 | Сложные интеграции (SAP, etc.) |
| Подписка | €9-29/мес | Доступ ко всем |

## Бизнес-модель

| Источник | Доход |
|----------|-------|
| Комиссия 20% | основной |
| Подписка Pro | €9-29/мес |
| Custom-сценарии | €100-2000 разовый |
| Featured-листинг | €49/мес |

## Конкуренция

| Платформа | Что продают | Чем отличаются |
|-----------|-------------|----------------|
| **n8n templates** | Бесплатные | Бесплатные, нет монетизации авторов |
| **Make.com templates** | Бесплатные | То же |
| **Авторская платформа** | Платные | Маркетплейс с авторами + комиссия |
| **Zapier templates** | Бесплатные | Закрытая платформа |

## Связь с микрофраншизой

Каждый сценарий = **микрофраншиза**: автор продаёт готовое решение и зарабатывает roalty + есть путь масштабирования.
