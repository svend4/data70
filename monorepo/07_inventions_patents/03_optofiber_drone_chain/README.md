# 03 — Оптоволоконная магистраль дронов 100 км — патентная карточка

**Категория:** A (Связь / Дроны)
**Новизна:** ★★★★
**Применимость:** ★★★
**Статус:** Сначала прототип звена

## Суть изобретения

Цепочка БПЛА-ретрансляторов, соединённых оптоволоконным кабелем на расстояние **100 км**. Куб-хабы (наземные точки) через каждые 20-30 км для замены батарей и регенерации сигнала.

**Ключевая инновация:** комбинация воздушной оптоволоконной связи с автономной БПЛА-платформой, способной работать без наземной инфраструктуры.

## Prior art

| Аналог | Особенность | Чем отличается |
|--------|-------------|----------------|
| **Tethered drones** | Привязные дроны (1-300 м) | 100 км через цепочку |
| **Оптоволокно военное** | Управление (короткое) | Магистраль (длинное) |
| **Mesh-сети дронов** (Wi-Fi) | Радиосвязь | Оптика — недоступно для глушения |
| **Спутниковая связь** (Starlink) | Глобально | Нулевая latency |

## Что заявляется

1. **Архитектура цепочки** — топология, плотность ретрансляторов
2. **Метод смены батарей** — посадка с пересостыковкой кабеля
3. **Куб-хаб** как наземный модуль с EDFA-усилителями
4. **Управление** — алгоритм координации позиций дронов

## Расчётные параметры

- Длина: 100 км
- Куб-хабов: 4-5 (каждые 20-30 км)
- Дронов-ретрансляторов: ~20
- Кабель: OS2 G.652D, ~250 кг/100 км
- Оптический бюджет: ~25 дБ
- Усилители: каждые 25-30 км

## Стратегия

1. **Прототип одного звена 5 км** — 1 дрон + 1 куб-хаб
2. **Партнёрство Fraunhofer FOKUS** (телеком)
3. **Заявка DPMA** после прототипа
4. **EIC Pathfinder** (€4M) — финансирование

## Применение

- Связь в зоне катастроф (гумбаны, ООН)
- Военная dual-use (защищённая связь)
- Сельхоз-мониторинг больших полей
- Подключение деревень в Африке (UNESCO, World Bank)
