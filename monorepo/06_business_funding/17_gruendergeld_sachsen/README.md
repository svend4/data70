# 17 — GründerGeld Sachsen

**Программа:** GründerGeld — региональная программа Саксонии
**Сумма:** до €25 000
**Условия:** для стартапов из Саксонии в первые 5 лет

## Что это

Региональная стипендия для саксонских стартапов. Меньше, чем EXIST, но **проще получить** и **не требует университетского контекста**.

## Условия

| Параметр | Значение |
|----------|----------|
| Регион | Саксония (Дрезден, Лейпциг, Хемниц) |
| Возраст компании | до 5 лет (или perpetration) |
| Команда | 1+ человек |
| Сумма | до €25K единовременно |
| Условия возврата | при определённых обстоятельствах |
| Дополнительные программы | SAB Sachsen (до €100K) |

## Подходящие проекты

Все продукты автора, особенно те, что **сразу запускаются**:
- Книга по PB
- Telegram-бот SozialHelper
- Маркетплейс n8n
- SmartOfficeProto

## Стратегия использования

GründerGeld — **первый «грант»** для тестирования системы:
1. Подача — относительно простая (5-10 страниц)
2. Срок — 3-4 нед. до решения
3. Использование — операционные расходы
4. После — опыт и отчётность для EXIST
