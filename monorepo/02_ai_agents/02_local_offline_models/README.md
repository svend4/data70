# 02 — Локальные нейросети оффлайн

**Тема каталога:** #14 (хаб, 18+ связей)
**Кластер A-J:** B
**Уровень готовности:** ★★★★★

## Что это

Запуск LLM **без интернета**, на локальном железе пользователя. Это не оптимизация, а **философия независимости и приватности**, лежащая в основе всех проектов автора.

## Почему это критично

1. **Приватность** — медицинские, юридические, личные данные не уходят в облако
2. **Соответствие GDPR** — для SaaS в Германии это сильный аргумент
3. **Стоимость** — нет API-биллинга, фиксированный CAPEX вместо OPEX
4. **Доступность** — работает в Африке, Азии, на старых устройствах
5. **Дёшево для пользователя** — модели на потребительском железе уже хороши

## Современный стек (2025-2026)

| Слой | Технология | Размер модели | Качество |
|------|-----------|---------------|----------|
| Quantization | GGUF Q4_K_M | -75% | -5-10% |
| Runtime | llama.cpp | ~5 МБ | производственный |
| Server | Ollama | ~50 МБ | удобство DevX |
| GPU | CUDA / Metal / Vulkan | — | критично для 70B+ |
| RAM | 16-32 ГБ | для моделей до 13B | бытовые ПК |
