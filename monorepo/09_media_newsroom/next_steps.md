# Следующие шаги — кластер 09_media_newsroom

## Краткосрочно (0-3 мес.)

1. **POC AI-newsroom**: SGB-новости из BSG, sozialgerichtsbarkeit.de
   - RSS → классификация → дайджест
   - Stack: Python + Haystack + Ollama
   - Срок: 2-4 нед

2. **Telegram-канал** «Sozialrecht Updates»
   - Автоматический дайджест 3 раза в неделю
   - Лидогенерация для книги по PB и SaaS

## Среднесрочно (3-12 мес.)

1. **AI-newsroom как SaaS** для региональных изданий
   - Цена: €299-999/мес. за издание
   - Партнёры: Sächsische Zeitung, Dresdner Neueste Nachrichten
   - Бюджет: €30-50K

2. **OSINT-сервис** для юристов и журналистов
   - Услуга «Раскопки темы за 24 часа»
   - Цена: €199-499 за тему
   - Платформа: Upwork + own site

3. **НейроПортал — юридический портал**
   - Использует AI-newsroom для постоянного обновления
   - Связано с `06_business_funding/20_neuroportal_content`

## Долгосрочно (12+ мес.)

1. **AI-newsroom для SkyMediaHub** — съёмка дронов автоматически становится новостями
2. **AI-newsroom для Pflege** — новости о ходах SGB, Pflegekasse, MDR
3. **B2B-API** — другие медиа подключают через API
