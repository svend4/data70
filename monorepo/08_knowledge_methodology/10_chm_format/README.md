# 10 — SLM + CHM формат

**Тема каталога:** #39
**Уровень готовности:** ★★★

## Что это

**CHM** (Compiled HTML Help) — старый формат Microsoft, до сих пор используемый для документации. Сжатые HTML-файлы в одном архиве, индексированные для поиска.

В контексте автора рассматривается как **формат для распространения SLM-обогащённого контента**:

- CHM-файл содержит структурированный контент
- Малая модель (SLM) внутри отвечает на запросы
- Один файл = полный курс/справочник + ИИ

## Концепция

```
.chm файл (~50 МБ)
├── Контент (HTML)
├── Индекс
└── SLM (~30 МБ Q4 Phi-3.5)

Открытие → клик по теме → SLM отвечает на вопросы
```

## Что отличает

| Решение | Размер | Особенность |
|---------|--------|-------------|
| **PDF** | 1-50 МБ | Без интерактивности |
| **EPUB** | 1-50 МБ | Чтение |
| **HTML-сайт** | разный | Нужен интернет |
| **CHM + SLM** | 50-100 МБ | Один файл, оффлайн, интерактив |

## Применение

- **Книга «Persönliches Budget»** — может выйти как CHM с встроенным SLM
- **Учебники** для развивающихся стран
- **Корпоративная документация** — компактно
- **Справочники** для медиков, юристов

## Современные альтернативы

- **Single-file webапp** (одна HTML-страница со всем)
- **PWA** (Progressive Web App) — оффлайн
- **Electron-приложение** — десктоп
