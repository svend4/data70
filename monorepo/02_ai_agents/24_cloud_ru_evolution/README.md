# 24 — Cloud.ru Evolution Stack AI

**Тема каталога:** #73
**Кластер A-J:** B
**Уровень готовности:** ★★

## Что это

**Cloud.ru Evolution Stack AI** — российская гибридная облачная платформа с поддержкой ИИ:
- Jupyter Lab (для исследований)
- Managed Kubernetes с GPU
- Реестр артефактов (модели, датасеты)
- S3-совместимое хранилище

Главная идея: **обучение в публичном облаке → запуск в частной среде**.

## Зачем это интересно автору

1. **Альтернатива Google Cloud / AWS / Azure** для российского/евразийского рынка
2. **Ценовая ниша** — дешевле западных аналогов в 2-3 раза
3. **Гибридное развёртывание** — критично для CareMate (медданные не уходят из частной сети)
4. **Связь с темой #18** (децентрализованные вычисления)

## Применение

- **CareMate Total для русскоязычных регионов** — обучать модели в Cloud.ru, разворачивать у клиента
- **SozialPlaner для UA/RU/KZ** — если расширяться на эти рынки
- **Альтернативная инфраструктура** на случай ограничений западных облаков

## Аналоги
