## Суть

Применение **тороидальной модели** (статья Франца Хермана «Zeitmaschine») к проектированию **микроэлектроники и чипов**:
- Два радиуса тора = два измерения вычислений
- Спиральные кривые вдоль тора как алгоритмы передачи данных
- Замкнутая поверхность даёт уникальные топологические свойства

## Зачем это нужно

Стандартные чипы — плоские (2D с многослойной литографией). Тороидальная топология даёт:
- Однородность путей (нет «края»)
- Меньшие задержки (любая точка достижима с любой)
- Естественную параллельность

## Что отличает

| Подход | Топология | Чем отличается |
|--------|-----------|----------------|
| **Классический чип** | 2D плоский | Стандартная литография |
| **3D чип** (HBM, NAND) | 2.5-3D стек | Слоённая структура |
| **Wafer-scale** (Cerebras) | 1 большая 2D плата | Огромная плата |
| **Тор** | 2D на 3D-поверхности | Замкнутая, без «краёв» |

## Что заявляется (возможно)

1. **Топология чипа** в форме тора (физическая или логическая)
2. **Алгоритмы маршрутизации** по спиральным путям
3. **Метод производства** изогнутой подложки
