# 04 — Оркестратор: ROS2/DDS + MQTT + Matter

**Уровень готовности:** ★★★★

## Что это

**Многоуровневый коммуникационный стек** для роя 15 роботов-зверей. Использует разные протоколы для разных задач.

## Стек

| Уровень | Протокол | Задача |
|---------|----------|--------|
| **Внутренняя связь робота** | ROS2 + DDS | Реалтайм-команды, сенсоры, моторы |
| **Между роботами** | DDS / Zenoh | Координация роя |
| **Лёгкая телеметрия** | MQTT | Состояние, статусы (low-bandwidth) |
| **Умный дом** | Matter | Интеграция с Philips Hue, Aqara, Apple Home |
| **Координация задач** | Contract Net Protocol | Распределение работы (см. `14_contract_net_protocol`) |
| **Веб-интерфейс** | WebSocket | Управление с телефона |

## Зачем такой комбо

Никакой один протокол не решает все задачи:
- ROS2 — стандарт робототехники, но тяжёлый
- MQTT — лёгкий, но без QoS гарантий ROS2
- Matter — для совместимости с потребительским умным домом
- Contract Net — для динамического распределения задач

## Архитектура коммуникаций

```
                  ┌── Веб (WS) ──┐
                  │              ↓
    ┌────────────────────────────────┐
    │     ОРКЕСТРАТОР CareMate        │
    │     (ROS2 + Contract Net)        │
    └─────┬──────┬──────┬─────┬──────┘
          │ DDS  │ MQTT │ Matter      
          ↓      ↓      ↓             
       Робот1  Сенсоры Hue лампы      
       Робот2          Aqara датч.    
        ...                           
```

## Сценарий: упавший человек

1. **Сенсор движения** (через MQTT) → Оркестратор: «обнаружено падение»
2. **Оркестратор** → Робот «Лиса» (через DDS): «иди к локации»
3. **Робот «Лиса»** прибывает, проверяет камерой
4. **Лиса** → Оркестратор (через DDS): «человек в сознании»
5. **Оркестратор** → Matter: «открыть дверь для скорой»
6. **Параллельно** — Snegir (Снегирь) приносит лекарство по запросу

## Технологии (open-source)

- **ROS2 Iron / Humble** — стандартный фреймворк
- **Zenoh** — облегчённая альтернатива DDS
- **Eclipse Mosquitto** — MQTT-брокер
- **OpenThread / Matter SDK** — стандарт умного дома
- **Home Assistant** — потребительский шлюз
