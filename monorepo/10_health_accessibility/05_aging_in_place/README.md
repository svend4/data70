# 05 — Старение в собственном жилье (Aging in Place)

**Уровень готовности:** ★★★

## Что это

**Aging in Place** — концепция и практика **старения в собственной квартире/доме**, без переезда в Pflegeheim. Главная альтернатива стационарному уходу.

В Германии это активный политический приоритет:
- Pflegekasse финансирует приспособление жилья
- Pflegegeld даёт деньги на ассистентов
- AAL Programme — для технологий
- Wohnraumanpassung — до €4 000

## Что нужно для Aging in Place

| Категория | Решения |
|-----------|---------|
| **Безопасность** | Notruf, мониторинг падения, дым/газ |
| **Мобильность** | Поручни, безбарьерный душ, расширенные двери |
| **Доступ к помощи** | 24/7 кнопка, видеосвязь с семьёй |
| **Когнитивная поддержка** | ИИ-помощник, напоминалки |
| **Социальный** | Волонтёры, соседская помощь, Pflegedienst |

## CareMate Total — решение

Все эти задачи решает CareMate Total. Это **главная ценностная пропозиция** для семей с пожилыми.

## Бизнес-модель «Aging in Place»

| Модуль | Цена |
|--------|------|
| Family-тариф CareMate | €14.90/мес |

- **Aging in Place + CareMate** — €30-100/мес + одноразовые

Экономия многократная, плюс достоинство и качество жизни.
