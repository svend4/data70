# 04 — Ассистивные технологии

**Уровень готовности:** ★★★

## Что это

Каталог **технологий, помогающих людям с инвалидностью** в повседневной жизни:

- Hilfsmittel (вспомогательные средства)
- Smart-home для людей с ОВЗ
- Голосовое управление
- Навигация в помещении
- Мониторинг здоровья
- Когнитивные помощники

## Категории

### Физическая помощь
- Treppenlift
- Электроинвалидное кресло
- Надувные подъёмные устройства
- Robotis (для приёма еды, гигиены)

### Сенсорная помощь
- Слуховые аппараты с ИИ
- Программы для слабовидящих (Be My Eyes)
- Сурдоперевод через ИИ

### Когнитивная помощь
- Напоминалки
- Диалоговые ИИ-агенты
- Тренировки памяти

### Smart-home
- Голосовые команды (Alexa, Google Home)
- Автоматизация (свет, климат)
- Сенсоры (падение, дым)

## Финансирование (DE)

| Категория | Финансирование |
|-----------|----------------|
| Hilfsmittel | Pflegekasse / Krankenkasse |
| Smart-home | до €4 000 от Pflegekasse |
| Robotis | пока нет статуса Hilfsmittel |
| Wohnraumanpassung | до €4 000 от Pflegekasse |
