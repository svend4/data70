# 12 — Коллективные профили ИИ

**Тема каталога:** #36 (приоритет А-12)
**Уровень готовности:** ★★★

## Что это

**Коллективные профили ИИ** — концепция, при которой несколько пользователей образуют **единый ИИ-профиль**, отражающий их общие предпочтения, ценности, цели.

Применения:
- Семья → семейный ИИ-профиль (для CareMate, образования, бюджета)
- Команда → командный профиль (для SozialPlaner, B2B)
- Сообщество → коллективная экспертиза (для НейроПортала)

## Что отличает

| Аналог | Тип | Чем отличается |
|--------|-----|----------------|
| **Личный профиль** | Индивидуальный | Не отражает общие интересы |
| **Команда в Slack** | Бизнес-коллектив | Без ИИ-личности |
| **Wikipedia** | Открытая коллабор. | Без персонализации |
| **Коллективные профили** | ИИ + коллектив | Адаптируется к группе |

## Архитектура

```
Пользователи (3-100) → запросы → 
   ↓
Profile Aggregator → коллективный профиль (вес каждого)
   ↓
Persona ИИ — генерирует ответы от имени группы
   ↓
Каждый пользователь видит общую базу + свою часть
```

## Применение в продуктах

- **CareMate Family** — семейный профиль (родители + дети + бабушка)
- **B2B MetaWorld** — корпоративный профиль (компания как ИИ-личность)
- **НейроПортал** — нишевой профиль (юристы, медики)
- **Цифровой Караван** — групповой профиль кочевников
