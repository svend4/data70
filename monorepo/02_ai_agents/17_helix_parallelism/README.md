## Что это

**Helix Parallelism** — техника параллелизации обучения/инференса от NVIDIA, использующая 3D-структуру (data + tensor + pipeline) на GPU-кластере.

**CXL (Compute Express Link)** — стандарт связи CPU-память-GPU с когерентностью кэшей. Позволяет создавать «пулы памяти» поверх нескольких устройств.

## Зачем это интересно

1. **Большие модели** — Helix позволяет тренировать модели, не помещающиеся в одну GPU
2. **CXL-пул** — десятки серверов делят общую память → дешевле, чем «всё в одном»
3. **Edge-ИИ** — те же принципы для распределения inference на нескольких устройствах

## Применение

В контексте автора это **долгосрочная R&D-тема**, не для прямой реализации, но как:
- Аргумент в EXIST/Horizon-заявках («понимаем будущее железа»)
- Архитектурный ориентир для НейроОС-инфраструктуры
- Связь с темой #6 (CiM) — все три направления формируют новое поколение ИИ-железа
