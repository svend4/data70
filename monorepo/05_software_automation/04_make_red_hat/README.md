# 04 — «Red Hat для no-code»

**Уровень готовности:** ★★★

## Что это

Бизнес-идея: как **Red Hat** заработал на **поддержке Linux** в enterprise (хотя Linux open-source), так можно заработать на **поддержке Make.com / n8n / Zapier** в enterprise.

## Аналогия

| Linux | No-code |
|-------|---------|
| Open-source | Open-source / freemium |
| Сложно настроить | Сложно настроить под бизнес |
| Кастомизация под клиента | Кастомизация под бизнес-процессы |
| Поддержка 24/7 нужна | Поддержка нужна |
| Сертификация админов | Сертификация автоматизаторов |
| **Red Hat зарабатывает** | **Эту нишу никто не занимает** |

## Сервисы

1. **Установка и настройка** — стек на инфраструктуре клиента (n8n self-hosted, Make.com Pro)
2. **Кастомизация** — кастомные коннекторы, узлы, плагины
3. **Поддержка 24/7** — SLA на работоспособность
4. **Обучение** — курсы для сотрудников клиента
5. **Сертификация** — «Certified n8n Architect»
6. **Аудит** — проверка существующих автоматизаций
7. **Мониторинг** — observability стек

- **Малый бизнес 5-50 чел.** — Starter

- **Корпорации 500+** — Enterprise
