# 14 — Аудио-Таро Сюрреализма — патентная карточка

**Категория:** D (Искусство / Концепции)
**Новизна:** ★★★★
**Применимость:** ★★
**Статус:** Колода + правила

## Суть

**Колода карт** на основе таро, но вместо визуальных образов — **аудио-образы и музыкальные сюжеты**. Каждая карта — короткий звуковой/музыкальный фрагмент, представляющий архетипический образ.

Сочетание:
- Структуры таро (Старшие и Младшие арканы)
- Сюрреалистической эстетики (необычные образы)
- Аудио-формата (вместо визуального)

## Что заявляется

1. **Концепция** — аудио-таро как новый формат
2. **Конкретная колода** — авторская подборка музыкальных образов
3. **Правила игры/гадания** — методология использования
4. **Цифровой формат** — приложение для воспроизведения карт

## Применение

- **Арт-объект** — выставки, инсталляции
- **Терапевтический инструмент** — арт-терапия
- **Цифровой продукт** — приложение
- **NFT-коллекция** — каждая карта как уникальный звуковой NFT
- **Образовательный инструмент** — для музыкального образования

## Защита

- **Авторское право** на конкретные звуковые композиции
- **Товарный знак** на название «Audio Tarot of Surrealism»
- **Дизайн-патент** на физическую колоду (если издаётся)

## Связи

- Арт-проекты как часть портфеля автора
- Возможная связь с НейроПорталом (нишевый портал об искусстве)

## Стратегия

1. Создание прототипной колоды (78 звуковых файлов)
2. Дизайн физических карт
3. Веб-приложение для онлайн-использования
4. Запуск на Kickstarter / Patreon

## Источники

- Концепция в архиве разговоров автора
- Связь с другими «геометрическими/символическими» проектами (Сефирот, Звезда в соте)
