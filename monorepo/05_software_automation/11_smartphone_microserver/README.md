# 11 — Смартфон как домашний микросервер

**Темы каталога:** #44, #81
**Уровень готовности:** ★★★★

## Что это

Использование **недорогого Android-смартфона ($100-200)** или старого устройства как:
- Центра управления умным домом
- Мини-NAS (хранилище)
- Сервера для локального ИИ
- Шлюза для IoT

## Что можно установить

| Сервис | Назначение |
|--------|-----------|
| **Termux** | Linux-подобная среда |
| **Ollama** (через Termux) | Локальный ИИ |
| **Home Assistant Client** | Управление умным домом |
| **Telegram-бот** | Уведомления и управление |
| **Nextcloud** | Облачное хранение |
| **Pi-hole** | Блокировка рекламы в LAN |
| **WireGuard** | VPN |
| **Самостоятельный mesh-узел** | Exo для распределённого ИИ |

## Зачем это нужно

1. **Стоимость** — $100-200 vs $500 за Raspberry Pi 5 + аксессуары
2. **Энергоэффективность** — смартфон на ARM ~5 Вт vs ~15 Вт RPi
3. **Встроенные батареи** — UPS «из коробки»
4. **Дисплей и сенсоры** — для GUI и автоматизаций
5. **Wi-Fi/BLE/4G** — все радио в одном устройстве
6. **Утилизация старого** — вторая жизнь для старых смартфонов

## Применения

- **Умный дом** в квартире пенсионера (CareMate-связка)
- **Мини-сервер** для фрилансера-кочевника
- **Семейный хаб** (Telegram-бот, домашний ИИ)
- **Класс программирования** для школ/университетов в развивающихся странах
