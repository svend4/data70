# 20 — НейроПортал — Content Platform + SaaS + Tokens

**Тип:** Content Platform + SaaS + Tokens
**MVP бюджет:** €150-200K
**Срок MVP:** 4-5 мес.
**Источник:** `part14_business_models.md` раздел 14.4

## Что это

Бизнес-модель **НейроПортала** — суперпроекта тематических ИИ-порталов нового интернета.

## Тарифы

| Источник | Механизм | Цена |
|----------|----------|------|
| **Читатель Free** | Базовый доступ | €0 |
| **Читатель Pro** | ИИ-поиск, дайджесты | €7.90/мес |
| **Автор/Издатель** | Публикация через инфоконвейер | €19.90/мес |
| **Нишевый портал (SaaS)** | White-label | €199-999/мес |
| **Токены за контент** (#82) | Авторам — за качественный контент | Revenue share 70/30 |
| **API** | RAG-поиск через API | Pay-per-query |

## Стратегия запуска (ниша → масштаб)

1. **Первая ниша:** юридический портал — законодательство + судебная практика + ИИ-анализ
2. **Вторая ниша:** медицинский портал
3. **Третья ниша:** техническая документация (DevOps, API)
4. **Платформа:** SaaS для создания нишевых порталов

## Путь к окупаемости

```
Мес 1-6:    MVP юридического портала, 1000 читателей
Мес 7-12:   5000 free + 300 Pro + 20 авторов = €5K MRR
Мес 13-18:  SaaS (white-label), 5 нишевых порталов = €12K MRR
Мес 19-24:  20 000 free + 1500 Pro + 15 порталов = €35K MRR
Мес 24-30:  API + токены = €50K+ MRR
Мес 30:     Безубыточность
```

## KPI

- Уник. читатели/мес
- Free → Pro конверсия (цель: 5-8%)
- Кол-во нишевых порталов (SaaS)
- Средний чек API-клиента
