# 10 — ИИ-автоматизация дропшипинга (Make.com)

**Тема каталога:** #49 (приоритет А-14)
**Уровень готовности:** ★★★

## Что это

Дропшипинг — модель электронной коммерции, при которой продавец **не хранит товар**, а покупает у поставщика по факту заказа. Ключевая роль автоматизации:

- Мониторинг остатков у поставщиков
- Автоматический заказ
- Уведомление клиентов
- Отслеживание доставки
- Бухгалтерия и налоги

## Архитектура автоматизации

```
Клиент заказывает
   ↓ webhook
Make.com / n8n сценарий:
   1. Проверка остатка у поставщика (API AliExpress/Spocket)
   2. Создание заказа у поставщика
   3. Сохранение в CRM
   4. Email клиенту с трекингом
   5. Запись в бухгалтерию
   6. Уведомление в Telegram
```
