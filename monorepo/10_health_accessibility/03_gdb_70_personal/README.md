# 03 — GdB 70: личный опыт автора

**Уровень готовности:** ★★★★★

## Что это

**GdB 70** — Grad der Behinderung 70%. Уровень инвалидности по немецкой системе. Связан с тяжёлыми ограничениями, но без полной нетрудоспособности.

В случае автора:
- Хронические заболевания
- Ограничения мобильности
- Потребность в ассистенции
- Pflegegrad 2-3

## Что даёт GdB 70

| Право | Описание |
|-------|----------|
| **Schwerbehindertenausweis** | Удостоверение |
| **Защита от увольнения** | Особая защита |
| **Дополнительный отпуск** | 5 рабочих дней/год |
| **Налоговые льготы** | Pauschbetrag €1 780/год |
| **Льготный проезд** | В отдельных случаях |
| **GFB-Status** | в Bundesagentur für Arbeit |
| **Ранний выход на пенсию** | при определённых условиях |

## Применение в продуктах

Опыт автора — **уникальное конкурентное преимущество** для CareMate Total:
- Знает изнутри, что нужно
- Может верифицировать сценарии
- Имеет credibility для маркетинга
- Понимает реальные риски и боль
