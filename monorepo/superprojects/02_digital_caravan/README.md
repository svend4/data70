# Суперпроект 2: Цифровой Караван

**Бюджет MVP:** €140-200K
**Срок MVP:** 5-6 мес
**Бизнес-модель:** Platform + Tokenomics
**MRR на 24 мес:** €25-35K
**Безубыточность:** ~28 мес

## Что это

Мобильная экосистема **цифровых кочевников**:
- Маркетплейс микрофраншиз
- Маркетплейс n8n-сценариев
- P2P-вычисления (Exo)
- Токеновая экономика (CARAVAN)
- Географический арбитраж
