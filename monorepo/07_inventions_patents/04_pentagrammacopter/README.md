# 04 — Пентаграммакоптер — патентная карточка

**Категория:** A (Аэродинамика)
**Новизна:** ★★★
**Применимость:** ★★★
**Статус:** Концепт

## Суть изобретения

5-роторный мультикоптер с **пентагональной геометрией** (звезда с 5 лучами). Альтернатива стандартным квадрокоптерам и гексакоптерам.

## Prior art

| Аналог | Особенность | Чем отличается |
|--------|-------------|----------------|
| **Quadcopter** (4 ротора) | Стандарт | Нет 5-симметрии |
| **Hexacopter** (6 роторов) | Resilience | Дороже |
| **Octocopter** (8 роторов) | Тяжёлый | Очень дорогой |
| **Tricopter** (3 ротора) | Дешёвый | Менее стабильный |

## Что отличает 5-симметрию

- Асимметрия даёт дополнительные степени контроля
- Полезная нагрузка в центре без затенения роторами
- Resilience при отказе одного ротора (4 продолжают)

## Что заявляется

1. **Геометрия рамы** — пентагональная, асимметричная
2. **Алгоритм управления** (нестандартный mixer для 5 моторов)
3. **Метод центрования полезной нагрузки** в геометрическом центре

## Стратегия

1. **Симуляция** в Gazebo / Webots
2. **Сравнение** с квадрокоптером той же массы
3. **Прототип** при подтверждении преимуществ
4. **Заявка** при ясной ниши

## Применение

- Полезная нагрузка в центре (камера)
- Городские полёты (стабильность)
- Образование (необычность как маркетинг)
