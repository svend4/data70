# 06 — LMS + SLM («AI-учитель в кармане»)

**Темы каталога:** #20, #34
**Уровень готовности:** ★★★★

## Что это

**LMS + SLM** = Learning Management System (типа Moodle) + Small Language Model (типа Phi-3.5).

Цель: **виртуальный учитель** для:
- Развивающихся стран (без стабильного интернета)
- Удалённых регионов
- Дешёвой массовой образования
- Курсов для людей с инвалидностью (CareMate-связка)

## Архитектура

```
Moodle (LMS) — структура курсов, оценки, прогресс
   ↓
Phi-3.5 (2 ГБ Q4) — оффлайн-учитель
   ↓
Контент: учебники, упражнения, видео
   ↓
Устройство: смартфон ($100) или Raspberry Pi
```

## Что отличает

| Решение | Особенность | Чем отличается |
|---------|-------------|----------------|
| **Khan Academy** | Видео + упражнения | Нет ИИ-чата |
| **ChatGPT-учитель** | Облако | Не работает оффлайн |
| **Duolingo** | Языки | Узкая специализация |
| **LMS+SLM** | Полный курс + ИИ оффлайн | Универсал, дёшев |

## Финансирование

| Программа | Сумма | Тема |
|-----------|-------|------|

## Применение в проектах автора

- **Образовательный TetraDrone-кит** — учитель робототехники
- **CareMate** — образовательный модуль для пожилых
- **Книги/курсы автора** — могут быть встроены в LMS+SLM
