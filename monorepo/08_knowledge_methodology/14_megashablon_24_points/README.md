# 14 — Мегашаблон карточки услуги (24 пункта)

**Уровень готовности:** ★★★★★

## Что это

**Авторский 24-пунктовый шаблон** для описания **любой социальной услуги** в Германии. Используется в каталоге 70 модулей.

Это **методологическая основа** работы с социальными услугами: единый формат описания, который позволяет:
- Сравнивать услуги
- Автоматически обрабатывать (RAG, классификация)
- Генерировать документы по шаблону
- Обучать пользователей

## 24 поля (обзор)

См. полное описание в `01_social_law/08_70_modules_catalog`.

| # | Поле | Назначение |
|---|------|-----------|
| 1 | Название (рус + нем) | Идентификация |
| 2-7 | Определение, цель, контекст, формат, регулярность | Описание |
| 8-14 | Стоимость (день/ночь/выходной/надбавки), мин. единица | Финансы |
| 15-17 | Правовая база, ведомство, документы | Право |
| 18-19 | Порядок подачи, сроки | Процедура |
| 20-22 | Право на Widerspruch, основания отказа, контраргументы | Защита |
| 23 | Судебная практика BSG | Прецеденты |
| 24 | Связи и эксклюзии | Композиция |

## Что отличает

| Аналог | Поля | Чем отличается |
|--------|------|----------------|
| **Wikipedia статья** | свободный текст | Нет структуры |
| **Юр. справочник** | формальный | Нет ИИ-связки |
| **Anwalt.de** | пользовательский | Нет финансов в едином формате |
| **24-пунктовый шаблон** | строгая структура | Готов для RAG/SaaS/книги |

## Применение

Это **базовый формат** для:
- Книги по PB
- SaaS SozialPlaner
- Telegram-бот
- Pflegekasse-отчётов
- Юридического ИИ-портала

## Расширение шаблона

Мегашаблон может быть адаптирован для других сфер:
- **Медицинские услуги** (DiGA-сертификаты, лекарства)
- **Образовательные программы** (курсы, обучение)
- **Юридические инструменты** (типы документов, прецеденты)
- **Бизнес-услуги** (категории, цены, регуляторика)
