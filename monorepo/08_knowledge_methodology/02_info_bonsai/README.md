# 02 — Инфо-Бонсай: персональная база знаний

**Темы каталога:** #46 (хаб 10+), #87 (расш.)
**Уровень готовности:** ★★★★

## Что это

**Инфо-Бонсай** — авторская концепция «второго мозга»: личной базы знаний, которая:
- Растёт постепенно (как бонсай)
- Подрезается / форматируется автором
- Сохраняет только релевантное
- Имеет чёткую структуру
- Доступна через ИИ-агента

Связана с **темой #87 «Умный домашний экзанет»** — расширение для всех видов документов.

## Что отличает от других PKM

| Решение | Особенность | Чем отличается |
|---------|-------------|----------------|
| **Roam Research** | Bidirectional linking | Bonsai — иерархическая |
| **Obsidian** | Markdown + плагины | Bonsai — встроенный ИИ |
| **Notion** | Облако + блоки | Bonsai — локальная, приватная |
| **Logseq** | Outliner | Bonsai — встроенная пирамида знаний |
| **Авторский Bonsai** | Локальная + ИИ-агент + Pyramid | Метафора + философия |

## Метафора бонсая

| Действие в бонсае | Действие в БЗ |
|-------------------|---------------|
| Подрезка ветвей | Удаление неактуального |
| Формирование кроны | Структурирование |
| Полив | Регулярные обновления |
| Защита от вредителей | Удаление шума |
| Ежегодный осмотр | Ревизия БЗ |

## Применение

- **Личная БЗ автора** — все 1105 разговоров организованы как Бонсай
- **Прототип для НейроОС** — пользователь имеет свой Бонсай в OS
- **Прототип для CareMate** — пожилые имеют Бонсай воспоминаний
