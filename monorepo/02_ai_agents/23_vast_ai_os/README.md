## Что это

**VAST AI OS** (от VAST Data) — операционная система для парка GPU и ИИ-агентов в дата-центрах. Три уровня:

1. **Data & Storage Layer** — единое хранилище для весов, данных, артефактов
2. **Compute & Agents Layer** — управление парком GPU + агентами
3. **Coordination & Orchestration Layer** — оркестрация задач между агентами

Это **инфраструктурный** аналог НейроОС: то же мышление, но для дата-центров.

## Чем интересна автору

1. **Эталон трёхуровневой архитектуры** (`13_three_level_arch`)
2. **Архитектурный ориентир** для НейроОС
3. **Для будущих B2B-продаж** — интеграция SozialPlaner с VAST для крупных клиентов
