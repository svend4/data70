# 05 — ВикиИИпедия (Портальный интернет)

**Темы каталога:** #21, #75
**Уровень готовности:** ★★★

## Что это

**ВикиИИпедия** — концепция нового типа онлайн-энциклопедии, в которой:
- Контент структурирован пирамидой знаний
- Каждая страница имеет встроенного ИИ-агента
- Читатель может запросить любой уровень подробности
- Авторы публикуют через инфоконвейер

**Портальный интернет** — расширение: каждая ниша имеет свой ИИ-портал (юридический, медицинский, технический, и т.д.).

## Чем отличается от Wikipedia

| Признак | Wikipedia | ВикиИИпедия |
|---------|-----------|-------------|
| Контент | Текст (статичный) | Адаптивный |
| Глубина | Одна для всех | По запросу |
| ИИ | Нет | Встроенный |
| Личный кабинет | Минимальный | Персональный путь |
| Монетизация | Donations | Подписки + токены |

## Архитектура

```
┌─────────────────────────────────────┐
│ Каждая страница:                     │
│   - Краткое введение (1-pager)      │
│   - ИИ-чат рядом                    │
│   - «Углубить» → следующий уровень  │
│   - «Упростить» → проще объяснить   │
│   - «Связано» → семантические связи │
└─────────────────────────────────────┘
```

## Применение

- **Юридический портал** (на основе SGB) — первый
- **Медицинский** (CareMate-связка)
- **Технический** (IFOS-документация)
- **Образовательный** (для развивающихся стран)
