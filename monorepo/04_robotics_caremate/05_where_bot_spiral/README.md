# 05 — ПылеСпираль-Бот (WHERE-Bot)

**Категория:** изобретение (см. `07_inventions_patents/07_where_bot_spiral`)
**Уровень готовности:** ★★★

## Что это

Применение принципа мягкого спирального робота (**WHERE-Bot**, IEEE RoboSoft 2025) для **двойной функции**: спираль одновременно используется для движения и сбора мусора.

## Принцип

```
Спираль из мягкого материала
   ↓ вращается → крутящийся «червь»
   ↓ заодно затягивает мусор внутрь
```

Спираль работает как:
- **Движитель** (как у штопора в почве)
- **Транспортёр** (захватывает частицы)
- **Сепаратор** (разделение по размеру)

## Чем интересно

1. **Двойное использование одного механизма** — простота, надёжность
2. **Мягкость** — безопасность для людей и питомцев
3. **Низкая стоимость** — нет жёсткой механики
4. **Универсальность** — работает на разных поверхностях (пол, ковёр, грязь, песок)

## Применения

- **Бытовой пылесос** — альтернатива Roomba (мягкий, тихий, дешёвый)
- **Уличная уборка** — сбор листьев в парках
- **Сельхоз** — сбор мусора с полей
- **Промышленность** — сбор стружки, пыли

## Аналоги

| Решение | Принцип |
|---------|---------|
| **Roomba** | Колёсная база + щётки + всос |
| **WHERE-Bot** (университет) | Спиральный мягкий робот (только движение) |
| **Авторская концепция** | Спираль для движения + сбора |
