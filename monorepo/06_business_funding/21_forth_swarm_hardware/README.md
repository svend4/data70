# 21 — Forth-Рой Hardware Kit + Licensing

**Тип:** Hardware Kit + Licensing + Services
**MVP бюджет:** €150-200K
**Срок MVP:** 6-8 мес.
**Источник:** `part14_business_models.md` раздел 14.5

## Что это

Бизнес-модель **Forth-Роя** — ультралёгкого ИИ для роботов и дронов.

## Источники дохода

| Источник | Механизм | Цена |
|----------|----------|------|
| **Стартовый кит «Рой-5»** | 5 дронов TetraDrone + станция + ПО | €2 500-5 000 |
| **Кит «Рой-20»** | 20 дронов + расширенная станция | €8 000-15 000 |
| **Подписка на ПО** | WILOS, ТСБ-обновления, аналитика | €49-199/мес |
| **OEM-лицензия Forth-интеллект** | Производителям роботов | €1-5 за устройство |
| **Обучение и консалтинг** | Для заказчиков | €500-2000/день |
| **Данные с дронов** | Агроаналитика | €2-5/га/сезон |

## Целевая аудитория

| Аудитория | Применение | TAM |
|-----------|-----------|-----|
| Агропредприятия | Мониторинг, опрыскивание | $5.5B к 2027 |
| Спасательные службы | Поиск, мониторинг | $1.2B к 2028 |
| Образование | Робототехника | $3.9B к 2028 |
| Логистика | Доставка | растущий |
| Оборона (dual-use) | Через отдельные контракты | EDF до €15M |

## Конкурентные преимущества

- **Стоимость дрона в рое:** €50-150 (vs €500-2000 у конкурентов)
- **Работа без интернета и GPS** — LoRa mesh + визуальная навигация
- **Открытые языки WILOS/ТСБ** — экосистема разработчиков
- **Одноразовые рамы TetraDrone** — биоразлагаемые, <€5

## Путь к окупаемости

```
Мес 1-8:    MVP, 3 пилота с фермами
Мес 9-14:   Первые продажи китов, 20 клиентов = €15K MRR
Мес 15-20:  OEM-лицензия Forth-интеллект, 2-3 партнёра
Мес 21-28:  50 клиентов + подписки + данные = €40K MRR
Мес 28-30:  Безубыточность
```

## KPI

- Кол-во проданных китов
- Кол-во активных дронов в поле
- OEM-лицензии (устройства/мес)
- Площадь обработанных полей (га/сезон)
