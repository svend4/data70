## Суть

Авторская математическая модель пропорций, объединяющая:
- **Золотое сечение φ ≈ 1.618**
- **Число π ≈ 3.14159**
- Их комбинации, степени, ряды

Применение к дизайну, архитектуре, музыке, ИИ-генерации образов.

## Известные «золотые» системы

| Система | Основа | Особенность |
|---------|--------|-------------|
| **φ-пропорции** | 1.618 | Витрувий, Парфенон |
| **Le Corbusier Modulor** | φ + чел. рост | Архитектура XX в. |
| **π-пропорции** | 3.14 | Реже используются |
| **Авторская система** | φ × π, φ^n × π^m | Новая комбинация |
