# 07 — Настольный робот-повар

**Тема каталога:** #79
**Уровень готовности:** ★★★

## Что это

**Промежуточный** между ручным кухонным комбайном и дорогим напольным роботом-поваром. Стоит на столешнице, имеет 2-3 мини-манипулятора, сменные насадки, встроенные весы и нагревательную поверхность.

Целевая аудитория — **средний класс с маленькими кухнями**.

## Спецификация (концепт)

| Параметр | Значение |
|----------|----------|
| Габариты | 50 × 40 × 60 см |
| Манипуляторы | 2-3 робо-руки с быстрозаменяемыми насадками |
| Насадки | Нож, ложка, лопатка, гриппер, погружной блендер |
| Весы | Встроенные с точностью 1 г |
| Нагрев | Индукционная зона + конвекция |
| ИИ | Локальный (Ollama) для распознавания продуктов и инструкций |
| Цена | €1500-3000 |

## Что отличает от существующих

| Решение | Тип | Цена | Размер |
|---------|-----|------|--------|
| **Thermomix** | Мульти-комбайн | €1500 | Стол |
| **Moley Robotic Kitchen** | Полный робот | €100K | Кухня |
| **Sous-Chef ($300)** | Простой ассистент | $300 | Стол |
| **Авторская концепция** | Робот-повар | €1500-3000 | Стол |

Ниша — **робот** (с манипуляторами и ИИ) на **столе** (без переоборудования кухни).

## Применения

- Семьи (быстрое приготовление под видеорецепты)
- Пожилые / люди с ограничениями (CareMate-связка)
- Студенты (дёшево + быстро)
- Маленькие кафе (мини-кухня)

## Связь с CareMate

Робот-повар может стать **физическим воплощением** CareMate Hub для пожилого человека:
- ИИ-диалог: «приготовим суп?»
- Робот сам режет и варит
- Голосовое управление
- Контроль порций для диабетика
