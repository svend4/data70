# 04 — Археология идей

**Уровень готовности:** ★★★★ (16-секционный документ)

## Что это

Авторская методология **ретроспективного OSINT** — раскопок собственных идей и архивов:

- Сбор всех материалов (разговоры, письма, заметки, документы)
- Классификация по темам, датам, проектам
- Извлечение «кладов» — ценных идей, которые могли быть забыты
- Воссоздание контекста и связей
- Создание «исторического слоя» в БЗ
