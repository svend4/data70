# 08 — B2B MetaWorld

**Источник:** разговор #705 и смежные ~8 разговоров
**Уровень готовности:** ★★★★

## Что это

**B2B MetaWorld** — амбициозная концепция трёхуровневой бизнес-платформы, объединяющей:
- LinkedIn (профессиональные связи)
- Alibaba (B2B-маркетплейс)
- SAP (ERP/закупки)
- MMORPG (геймификация для вовлечения)

## Трёхуровневая архитектура

```
СТРАТЕГИЧЕСКИЙ — экосистемы, инвестиции, M&A
       ↓
ТАКТИЧЕСКИЙ    — партнёрства, тендеры, кластеры
       ↓
ОПЕРАТИВНЫЙ    — сделки, заказы, логистика
```

Это пример **мета-паттерна «Три уровня всюду»** (см. `02_ai_agents/13_three_level_arch`).

## API-стек

| Стандарт | Назначение |
|----------|-----------|
| **OAuth/OIDC** | Аутентификация компаний |
| **GraphQL** | Гибкие запросы данных |
| **REST** | Стандартная интеграция |
| **PEPPOL** | Электронные счета EU |
| **EDI** (X12, EDIFACT) | Корпоративный документооборот |
| **OpenSearch** | Поиск компаний/продуктов |
| **WebSockets** | Realtime-уведомления |

## Compliance

- **GDPR** — персональные данные
- **AGG** — антидискриминация
- **B2B Regulation EU** — общие правила
- **DSA** (Digital Services Act) — для маркетплейсов

## Геймификация

| Элемент | Применение |
|---------|-----------|
| **Уровни компаний** | Bronze → Silver → Gold → Platinum (на основе сделок) |
| **Достижения** | «100 успешных сделок», «5 стран» |
| **Гильдии** | Кластеры компаний по нише |
| **Квесты** | Совместные тендеры |
| **Турниры** | Хакатоны для компаний |
