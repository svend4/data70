# 08 — NeuroB2B

**Тема каталога:** #38 (приоритет А-12)
**Уровень готовности:** ★★★★

## Что это

**B2B-нейросеть** — платформа для бизнес-взаимодействий с встроенным ИИ. Объединяет функции LinkedIn (профессиональная сеть), Alibaba (B2B-маркетплейс), SAP (ERP) и MMORPG (геймификация).

В материалах автора это связано с концепцией **B2B MetaWorld** (`06_business_funding/08_b2b_metaworld`).

## Архитектура — три уровня

| Уровень | Что |
|---------|-----|
| **Оперативный** | Сделки, заказы, логистика, оплата |
| **Тактический** | Партнёрства, тендеры, кластеры |
| **Стратегический** | Экосистемы, инвестиции, M&A |

Это пример мета-паттерна «Три уровня всюду» (см. `02_ai_agents/13_three_level_arch`).

## Состав

| Модуль | Функция |
|--------|---------|
| Профили компаний | LinkedIn-стиль с верификацией |
| Каталог продуктов | Alibaba-стиль |
| ERP-интеграция | SAP / NetSuite через PEPPOL/EDI |
| ИИ-агенты | Поиск партнёров, переговоры, рекомендации |
| Геймификация | Достижения, рейтинги, кооперативные миссии |
| Compliance | GDPR, AGG, B2B-регуляторика EU |

## Технологии

- **Frontend:** Next.js + React + TypeScript
- **Backend:** Node.js (Fastify) + PostgreSQL
- **Identity:** OAuth/OIDC (Keycloak)
- **API:** GraphQL для гибких запросов + REST для интеграций
- **Документообмен:** PEPPOL + EDI (X12, EDIFACT)
- **Аналитика:** ClickHouse
- **ИИ:** Claude API + локальный
