# 11 — Метод Master Dossier

**Уровень готовности:** ★★★★★

## Что это

**Метод Master Dossier** — методологический подход к ведению **полного личного дела** в любой жизненно важной области (соцправо, медицина, финансы, образование).

## Принципы

1. **Один источник правды** — всё в одном месте
2. **Версионирование** — каждое изменение фиксируется
3. **Машиночитаемость** — YAML/JSON для автоматизации
4. **Полнота** — все ведомства, все документы, все сроки
5. **Текущая стратегия** — не только история, но и план действий

## Связь с пирамидой знаний

Master Dossier — **практический уровень** пирамиды:
- Стратегия: «отстоять права»
- Тактика: «параллельная подача»
- Оперативное: «следующий Antrag»
- **Практическое**: реальные документы, переписка, копии

## Применение

| Сфера | Master Dossier |
|-------|----------------|
| **Соцправо** | См. `01_social_law/07_master_dossier` |
| **Медицина** | Карта здоровья, история болезни, лекарства |
| **Финансы** | Бюджет, счета, налоги |
| **Образование** | Дипломы, курсы, прогресс |
| **Бизнес** | Контракты, документация, проекты |

## Что отличает от обычной папки

| Признак | Папка | Master Dossier |
|---------|-------|----------------|
| Структура | Хаотичная | Стандартизированная |
| Версии | Перезапись | Полная история |
| Сроки | Не отслеживаются | Активные дедлайны |
| Стратегия | Нет | Есть текущий план |
