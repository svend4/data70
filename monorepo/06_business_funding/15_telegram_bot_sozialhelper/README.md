# 15 — Telegram-бот SozialHelper

**Цена:** Freemium (€4.90/мес. за генерацию документов)
**Бюджет:** €200-500
**Срок:** 4-6 недель
**Прогноз:** €1-3K/мес. через 3 мес.

## Что это

**Telegram-бот** для людей, нуждающихся в помощи с немецкой соцсистемой. Реализует RAG-поиск по корпусу SGB-документов автора + AI-генерацию шаблонов.

## Функции

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие + языковой выбор (DE/RU) |
| `/sgb` | Поиск по параграфам SGB |
| `/widerspruch` | Генератор Widerspruch (Pro) |
| `/klage` | Генератор Klage (Pro) |
| `/budget` | Калькулятор Persönliches Budget |
| `/services` | Каталог 70 модулей соцуслуг |
| `/cases` | Поиск BSG-прецедентов |
| `/compare` | Сравнение РФ-DE соцуслуг |
| `/help` | Помощь |
| `/upgrade` | Переход на Pro |

## Тарифы

| Тариф | Цена | Что входит |
|-------|------|-----------|
| Free | €0 | Поиск SGB, базовые ответы, лимит 20 запросов/день |
| Pro | €4.90/мес | Безлимит, генерация документов, экспорт PDF |
| Team | €19/мес | 5 пользователей, общая база |

## Технологии

- **Aiogram** (Python) — Telegram-бот фреймворк
- **Ollama** + Llama 3.1 8B Q4 — локальный ИИ (приватно!)
- **ChromaDB** — векторная БД (RAG)
- **Stripe Subscription** — оплата Pro
- **PostgreSQL** — пользовательская БД
- **Hetzner VPS** (€10-20/мес) — хостинг
