# 11 — Agro-Scout: сельхоз-мониторинг

**Уровень готовности:** ★★★

## Что это

Дрон для сельхоз-мониторинга: NDVI-съёмка, поиск вредителей, мониторинг роста растений, точное земледелие. Часть SkyMediaHub-портфеля.

## Спецификация

| Параметр | Значение |
|----------|----------|
| Платформа | TetraDrone или малый VTOL |
| Камера | RGB + NIR (для NDVI) или мультиспектральная |
| Время полёта | 25-45 мин |
| Площадь обработки | 50-200 га / полёт |
| Цена для фермера | €500-2000 за дрон |
| Услуга | €2-5 / га / сезон |

## Бизнес-модель

1. **Продажа дронов фермерам** — €500-2000
2. **Услуга «дрон по запросу»** — фермер платит за каждый облёт
3. **Подписка на ПО** — €49-199/мес (анализ данных, отчёты)
4. **Данные** — агрегированная карта NDVI региона для агропромышленности
