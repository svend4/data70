# 07 — ПылеСпираль-Бот — патентная карточка

**Категория:** B (Робототехника)
**Новизна:** ★★★★
**Применимость:** ★★★
**Статус:** Концепт

## Суть изобретения

Применение принципа **мягкого спирального робота** (WHERE-Bot, IEEE RoboSoft 2025) для **двойной функции**: спираль одновременно используется для **движения и сбора мусора**.

**Ключевая инновация:** двойное использование одного механизма, что снижает массу, стоимость и риск отказа.

## Prior art

| Аналог | Особенность | Чем отличается |
|--------|-------------|----------------|
| **WHERE-Bot** (academic) | Только движение | + сбор мусора |
| **Roomba** | Колёсная база + щётки | Полностью спираль |
| **Soft robots** в целом | Различные применения | Двойное использование |

## Что заявляется

1. **Конструкция спирали** с интегрированной системой сбора
2. **Алгоритм управления** при двойном использовании
3. **Геометрия захвата** мусора в витки
4. **Метод выгрузки** на станции

## Применение

- Бытовой пылесос (тихий, безопасный)
- Уличная уборка (парки)
- Сельхоз (сбор мусора с полей)
- Промышленность (сбор стружки)
