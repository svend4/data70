# 10 — Молния-2: VTOL разведывательный дрон

**Уровень готовности:** ★★★

## Что это

VTOL-дрон (Vertical Take-Off and Landing) для разведки. Сочетает вертикальный взлёт мультикоптера с горизонтальным полётом самолёта — экономия энергии при длительных миссиях.

## Спецификация (концепт)

| Параметр | Значение |
|----------|----------|
| Размах | 1.5-2 м |
| Взлётный вес | 3-7 кг |
| Время полёта | 60-120 мин |
| Дальность | 50-100 км |
| Полезная нагрузка | 500 г - 1.5 кг |
| Скорость | 80-120 км/ч (горизонт), 5-10 м/с (вертикаль) |

## Применения

- Разведка местности, картография
- Сельхоз-мониторинг больших полей
- Спасательные операции — поиск пропавших
- Морская/прибрежная разведка
- Военная разведка (dual-use)
