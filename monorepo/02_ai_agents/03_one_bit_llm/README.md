## Что это

**1-Bit LLM** — нейросети, в которых веса представлены 1 битом (или 1.58 бит, троичная логика {-1, 0, 1}). Главные представители: BitNet b1.58 (Microsoft Research), HBitLinear.

Преимущества:
- В 8-16 раз меньше памяти, чем FP16
- В 6 раз меньше энергии
- Возможность запуска без GPU
- Работа на микроконтроллерах (ESP32-S3, STM32)

## Почему это важно для проектов автора

1. **Forth-Рой** — каждый дрон/робот несёт 1-Bit модель на ESP32
2. **Edge-ИИ** — модели в смартфонах, IoT, бытовой технике
3. **Глобальный охват** — работа в развивающихся странах
4. **CareMate** — приватность медданных через крошечные локальные модели

## Технологии

| Подход | Размер веса | Где работает |
|--------|-------------|--------------|
| FP32 | 32 бита | Облако |
| FP16 | 16 бит | GPU |
| Q4_K_M (GGUF) | 4 бита | Бытовые ПК |
| Q2_K | 2 бита | RPi, старое железо |
| BitNet b1.58 | 1.58 бит | MCU, edge |
| Бинарные сети | 1 бит | Минимальное железо |

## Эталонная цепочка

```
обычная LLM (Llama 3.1 8B)        — ~16 ГБ FP16
   ↓ квантизация
GGUF Q4 (та же модель)            — ~5 ГБ
   ↓ агрессивная квантизация
GGUF Q2                            — ~3 ГБ
   ↓ переобучение в 1.58 бит
BitNet 8B b1.58                   — ~2 ГБ, в 6 раз меньше энергии
   ↓ дистилляция
Forth-интеллект на ESP32         — <4 КБ, см. 07_inventions_patents/22
```

- Качество 1.58-bit моделей пока не на уровне FP16 на сложных задачах

- Library support пока ограничен (BitNet.cpp)
