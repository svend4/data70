# 07 — TiddlyWiki как PKM-инструмент

**Уровень готовности:** ★★★

## Что это

**TiddlyWiki** — необычный PKM-инструмент: вся вики хранится в **одном HTML-файле**, который содержит и контент, и движок. Открывается в браузере, работает оффлайн.

В контексте автора TiddlyWiki — **один из вариантов** реализации Инфо-Бонсая для пользователей, не желающих инсталлировать сложный софт.

## Что отличает

| Решение | Хранение | Особенность |
|---------|----------|-------------|
| **Notion** | Облако | Удобно, но не приватно |
| **Obsidian** | Локально, Markdown | Нужна установка |
| **Roam Research** | Облако | Платный |
| **TiddlyWiki** | 1 HTML-файл | Полная мобильность |

## Применение

1. **Учебная пирамида знаний** — для учеников курса автора
2. **Подаренная БЗ** — для пользователей TelegramBot SozialHelper
3. **Прототип Bonsai** — для тех, кто не хочет ставить Ollama

## Связи

- `02_info_bonsai` — концептуальная база
- `01_pyramid_4_levels` — может быть структурирован под пирамиду
- Может быть распространяется как одноразовый бонус с книгой по PB

## Стратегия

1. **Авторская конфигурация TiddlyWiki** — с готовой пирамидой и шаблонами
2. **Распространять как бонус** к книге по PB (€19-29)
3. **Plugin-набор** — авторские дополнения для PKM
