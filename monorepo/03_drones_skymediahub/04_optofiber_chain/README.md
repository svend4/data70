# 04 — Оптоволоконная магистраль дронов 100 км

**Категория:** изобретение (см. `07_inventions_patents/03_optofiber_drone_chain`)
**Уровень готовности:** ★★★★★ (расчёт завершён)

## Что это

**Цепочка БПЛА-ретрансляторов**, соединённых **оптоволоконным кабелем** на расстояние 100 км. Куб-хабы (наземные точки опоры) через каждые 20-30 км.

Назначение:
- Защищённая связь там, где нет инфраструктуры (фронт, удалённые регионы)
- Передача больших объёмов данных (видео 4K со множества дронов одновременно)
- Альтернатива спутниковому интернету (Starlink) с фиксированной задержкой

## Параметры расчёта (готовы)

| Параметр | Значение |
|----------|----------|
| Длина магистрали | 100 км |
| Кол-во куб-хабов | 4-5 (через 20-30 км) |
| Кол-во дронов-ретрансляторов | ~20 (по 4-5 на пролёт) |
| Тип кабеля | OS2 G.652D (стандартный single-mode) |
| Масса кабеля | ~2.5 кг/км → 250 кг/100 км |
| Оптический бюджет | 0.25 дБ/км → ~25 дБ на 100 км |
| Усилители (EDFA) | каждые 25-30 км |
| Энергия дрона-ретранслятора | 20-30 Вт |
| Точность посадки | <0.5 м (для пересостыковки) |

## Архитектура

```
Куб-хаб A (наземный)
      ↑
   дрон-ретранслятор 1 (висит на 100-300 м)
      ↑ оптоволокно
   дрон-ретранслятор 2
      ↑
   дрон-ретранслятор 3
      ↑
Куб-хаб B (наземный, +20 км)
```

Каждый дрон висит вертикально, кабель подвешен. При посадке для замены батарей — другой дрон поднимается на его место.

## Применения

| Сценарий | Аудитория |
|----------|----------|
| Связь в зоне катастроф | Гумбаны, Красный крест, ООН |
| Связь во фронтовой зоне | Военные (dual-use) |
| Сельхоз-мониторинг больших полей | Агро-кооперативы |
| Подключение деревень в Африке | Гранты (UNESCO, World Bank) |
| Временная связь на стройках | Crawler / поэтапные стройки |

## Конкуренты / альтернативы

| Решение | Расстояние | Latency | Цена |
|---------|-----------|---------|------|
| **Оптомагистраль дронов** | 100 км | <1 мс | средняя |
| **Starlink** | глобально | 25-50 мс | низкая |
| **Микроволны (Ubiquiti)** | 30 км | 1 мс | низкая |
| **GSM/LTE/5G** | от вышки | низкая | низкая |
| **Подводный кабель** | трансконтинент | низкая | очень высокая |

Ниша — **большая полоса (видео!) + низкая latency + разворачивается за часы**.
