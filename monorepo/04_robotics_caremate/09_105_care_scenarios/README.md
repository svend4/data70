**Приоритет:** А (13 — можно начинать сейчас)
**Уровень готовности:** ★★★★★

## Что это

**Библиотека из 105 готовых сценариев** для CareMate Hub и подобных систем заботы о пожилых, инвалидах, хронически больных.

Каждый сценарий — **триггер + действие + правовая привязка** (если есть).

## Структура сценария

```yaml
scenario:
  id: fall_detection
  trigger:
    - sensor: motion
      condition: sudden_horizontal_acceleration
    - sensor: camera
      condition: person_lying_motionless
  action:
    - voice_query: "Sind Sie in Ordnung?"
    - wait_seconds: 30
    - if_no_response:
        - call_family
        - unlock_door
        - notify_pflegedienst
  legal:
    sgb: "§ 14 SGB XI"
    pflegegrad_min: 2
  pflege_module: 5  # из 70-модульного каталога
```

## Примеры конкретных сценариев

1. **Падение** — голос-опрос, через 30 сек звонок семье
2. **Пропуск приёма лекарства** — напоминание + эскалация
3. **Высокая температура** — измерение + уведомление врача
4. **Открыта плита 2 ч** — отключение + предупреждение
5. **Сосед-волонтёр** — связь через социальную MMORPG
6. **Прогулка не состоялась** — напоминание о важности активности
7. **Нет движения 6 ч** — мягкая проверка
8. **Холодильник пустой** — заказ продуктов
9. **Звонок мошенника** — анализ голоса + предупреждение
