# 07 — Bytebot — RPA-автоматизация рабочего стола

**Тема каталога:** #12 (приоритет А-13)
**Уровень готовности:** ★★★★

## Что это

**Bytebot** — open-source RPA-инструмент. Запись и воспроизведение действий на рабочем столе: клики, ввод, навигация. Современная альтернатива закрытым решениям (UiPath, Blue Prism).

## Применения в проектах автора

| Сценарий | Что автоматизирует |
|----------|--------------------|
| Sozialamt-портал | Заполнение онлайн-форм |
| Krankenkasse-портал | Скачивание Bescheid |
| Erstattung-формы | Подача расходов |
| Pflegekasse | Отчёты, заявления |
| Любой no-API сервис | Через GUI |

## Технологический стек

- **Python + PyAutoGUI** — базовая автоматизация
- **Selenium / Playwright** — для веб
- **OCR (Tesseract)** — распознавание скриншотов
- **AI-vision** — модель видит экран и решает, куда кликать

## Архитектура

```
┌────────────────────────────────┐
│  Запись макроса (training)     │
│  - Скриншоты + клики           │
│  - Пользователь демонстрирует  │
└──────────┬─────────────────────┘
           ↓
┌────────────────────────────────┐
│  Bytebot модель                │
│  Вход: скриншот + цель         │
│  Выход: координаты клика       │
└──────────┬─────────────────────┘
           ↓
┌────────────────────────────────┐
│  Воспроизведение               │
│  - Открыть приложение          │
│  - Выполнить шаги              │
│  - Адаптация к новому виду     │
└────────────────────────────────┘
```

## Применение в SozialPlaner

```
1. Пользователь логинится в портал Sozialamt Dresden
2. SozialPlaner знает структуру Antrag
3. Bytebot-сценарий заполняет форму
4. Сохраняет, отправляет
5. Подтверждение возвращается обратно в SozialPlaner
```

Это **критическая фишка** — для пользователя SaaS не нужно знать, как пользоваться государственным порталом.
