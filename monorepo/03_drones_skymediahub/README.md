# 03 — Дроны и SkyMediaHub Bavaria

**~50 разговоров · 4.6 МБ текста · уровень готовности 4/5**

## О чём этот кластер

Кластер собирает все материалы по **БПЛА (беспилотники), медиа-экосистеме SkyMediaHub Bavaria и dual-use** инженерным концепциям. Это второй самый коммерчески перспективный блок после соцправа.

Ключевые активы:
- **13 проектов дронов** с техспецификациями
- **Инженерные расчёты** для оптоволоконной магистрали 100 км
- **Составная 4K-камера** из 28 матриц 600×600 (расчёт)
- **WILOS** — формальный язык управления роем (DSL)
- **Письма-запросы** на немецком в Fraunhofer, TechHUB SVI, Bundeswehr
- **Концепция конверсии** автопромышленности под UGV

## Зачем это нужно

1. **Грантовые программы** — Bayern Innovativ (€200K), Horizon Europe EIC Accelerator (€2.5M)
2. **Партнёрства** — уже есть письма в Fraunhofer и TechHUB SVI Sachsen
3. **Рынок агродронов** — $5.5 млрд к 2027 (CAGR 22%)
4. **Dual-use** — финансирование ЕС активно, особенно после 2022
