## Суть

**ТСБ** — DSL для описания **физических действий роботов**. Универсальный язык движений, генерирует PWM-последовательности для конкретных моторов и манипуляторов.

Аналог для роботов того, чем для дронов является WILOS (`09_wilos_dsl`).

## Иерархия

```
WILOS  ───────────► «Где должен быть»  (стратегия роя)
   ↓
ТСБ    ───────────► «Как двигаться»     (тактика робота)
   ↓
Forth  ───────────► «Команды моторам»   (операции)
```

## Чем отличается

| Аналог | Уровень | Чем отличается |
|--------|---------|----------------|
| **ROS2 trajectory msgs** | Низкий | ТСБ — выше, декларативный |
| **MoveIt! action** | Среднего | ТСБ — для разных конструкций |
| **OpenSesame Movement** | Academic | ТСБ — production |

## Что заявляется

1. **Грамматика языка** для физических действий
2. **Метод трансляции** в PWM/действия моторов разных конструкций
3. **Универсальность** — работает с разным числом конечностей (см. `04_robotics_caremate/11_robot_limbs_principle`)
4. **Композиция** движений — объединение в сложные действия

## Пример ТСБ-кода

```tsb
movement pickup_object {
  prerequisite:
    object_visible: true
    distance_to_object: between(20cm, 50cm)
    
  steps:
    - extend_manipulator: target=object
    - close_grip: pressure=medium
    - lift: height=10cm
    - retract_manipulator
    
  postcondition:
    object_in_grip: true
    
  fallback:
    if extend_manipulator_failed:
      - approach: distance=10cm
      - retry pickup_object
}
```

## Применение

- Forth-Рой — координация движений роботов в рое
- 15 роботов-зверей — общий язык для всех
- Образование — учебный язык робототехники
- OEM-лицензия для производителей роботов
