# 01 — SkyMediaHub Bavaria

**Зонтичный проект кластера**
**Уровень готовности:** ★★★★

## Что это

**SkyMediaHub Bavaria** — экосистема, объединяющая:
- 13 проектов дронов (от одноразовых до тяжёлых платформ)
- AI-newsroom (генерация контента из съёмки дронов)
- Партнёрство с Fraunhofer и TechHUB SVI
- Dual-use (гражданская + оборонная) направленность

**География:** изначально Bavaria (Bayern Innovativ грант), масштабирование на DACH и EU.
