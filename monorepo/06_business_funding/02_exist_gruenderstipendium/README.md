# 02 — EXIST-Gründerstipendium

**Программа:** EXIST — Бизнес-стартап Министерства экономики и энергетики Германии (BMWi)
**Сумма:** до €150 000 в год
**Сроки:** до 12 месяцев
**Условия:** для команд из вузов

## Что это

EXIST-Gründerstipendium — главная немецкая программа поддержки технологических стартапов. Финансирует **зарплаты основателей и команды** в течение 12 месяцев, плюс операционные расходы.

## Структура финансирования

| Статья | Размер |
|--------|--------|
| Зарплата основателю с PhD | €3 000/мес |
| Зарплата основателю с дипломом | €2 500/мес |
| Зарплата техсотруднику с дипломом | €2 000/мес |
| Зарплата студенту-сотруднику | €1 000/мес |
| **Лимит зарплат** | до 4 чел., €100K/год |
| **Сабсиди расходов** | до €30K/год |
| **Дети (надбавка)** | €150/мес/ребёнок |

## Ключевые требования

1. **Связь с университетом** — есть EXIST-«хаб» (TU Dresden имеет такой)
2. **Минимум 1 человек с дипломом университета**
3. **Бизнес-план** — 30-50 страниц
4. **Ментор от вуза** — научный сотрудник как «куратор»
5. **Не более 5 лет** с момента получения диплома команды

## Лучшие проекты для EXIST

Из портфеля автора подходят:
1. **SozialPlaner SaaS** — самое быстрое прохождение (готовый контент + понятная боль)
2. **НейроОС / Internet Function OS** — техническое + ИИ
3. **CareMate Total** — социальное + ИИ + AAL-партнёрства

## Документы для заявки

1. **Бизнес-план** (30-50 стр.)
2. **CV основателей**
3. **Письмо от ментора** TU Dresden
4. **Финансовый план** (3 года)
5. **Маркетинговый план**
6. **Технологический план**
7. **Командный план** (роли, доли)

## Стратегия подачи

1. **Связь с TU Dresden** — Center for Entrepreneurship
2. **Поиск ментора** — профессор по теме (право/ИИ/робототехника)
3. **Поиск ко-фаундера** — 1-2 студента/выпускника
4. **Подготовка бизнес-плана** — 4-8 недель
5. **Подача заявки** — постоянный приём

## Калькуляция бюджета

Пример: SozialPlaner SaaS, команда 3 чел.

| Статья | €/мес | €/год |
|--------|-------|-------|
| Основатель (диплом) | 2 500 | 30 000 |
| Co-founder (диплом) | 2 500 | 30 000 |
| Студент-разработчик | 1 000 | 12 000 |
| **Зарплаты** | **6 000** | **72 000** |
| Hosting + API | 500 | 6 000 |
| Маркетинг | 1 000 | 12 000 |
| Юр. услуги | 500 | 6 000 |
| Прочее | 500 | 6 000 |
| **Расходы** | **2 500** | **30 000** |
| **ИТОГО** | **8 500** | **102 000** |

Из EXIST покрывается: €100K + €30K = €130K. Остаток €28K — через INVEST или Bootstrap.
