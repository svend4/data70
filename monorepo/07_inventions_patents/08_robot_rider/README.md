# 08 — Гибридный робот-всадник — патентная карточка

**Категория:** B (Робототехника)
**Новизна:** ★★★
**Применимость:** ★★★
**Статус:** Концепт

## Суть изобретения

**Двухмодульный робот**: верхний модуль с манипуляторами и ИИ, нижний модуль на колёсах/ногах/гусеницах. Соединяются через **универсальный разъём** — открытый стандарт.

**Ключевая инновация:** модульная конструкция роботов с открытым стандартом разъёма (как USB-C для электроники).

## Prior art

| Аналог | Тип | Чем отличается |
|--------|-----|----------------|
| **Boston Dynamics Atlas** | Цельный | Не модульный |
| **Boston Dynamics Spot + Arm** | Близко к идее | Закрытый стандарт |
| **Tesla Optimus** | Цельный | Не модульный |
| **Industrial robots** (KUKA, ABB) | Стационарные | Не мобильные |

## Что заявляется

1. **Открытый стандарт разъёма** между модулями (механика + питание + связь)
2. **Концепция двухмодульной конструкции** с заменяемыми носителями
3. **Архитектура управления** при смене носителя без перепрошивки
4. **Метод калибровки** при сборке/разборке

## Зачем открытый стандарт

Аналогия: USB-C → разные производители → совместимость.

| Производитель торса | Производитель носителя |
|---------------------|------------------------|
| Boston Dynamics торс | Любой носитель с разъёмом |
| Custom торс | Любой Spot-подобный |
| Простой Pi-based торс | Колёсная платформа |

## Применение

| Сценарий | Носитель | Торс |
|----------|----------|------|
| Склад | Колёсный | Гриппер |
| Спасение | 4-ногий | Камера + рука |
| Промышленность | Гусеничный | Сварочный |
| Уход за пожилыми | Тихий колёсный | Мягкие манипуляторы |

## Стратегия

1. **Спецификация разъёма** (механика, электрика, связь)
2. **Open standard** — публикация на GitHub под Apache 2.0
3. **Контакт** с Boston Dynamics, NVIDIA Isaac, OpenAI Robotics
4. **Подача** на отдельные нюансы реализации
