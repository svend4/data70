# 09 — Мобильность и транспорт

**Уровень готовности:** ★★★

## Что это

Вопросы **передвижения** для людей с инвалидностью в Германии:

- Sonderparkausweise (специальные парковки)
- Bahncard для инвалидов (BC25/BC50/BC100 с льготами)
- Schwerbehindertenausweis с метками (G, aG, B)
- Mobilitätshilfen (электроинвалидное кресло, Treppenlift)
- Transport-сервисы (Krankentransport)

## Метки в Schwerbehindertenausweis

| Метка | Что значит |
|-------|-----------|
| **G** | Erhebliche Gehbehinderung — пешеходные ограничения |
| **aG** | Außergewöhnliche Gehbehinderung — крайние пешеходные |
| **B** | Berechtigung zur Mitnahme einer Begleitperson — провожатый |
| **H** | Hilflos — нуждается в помощи |
| **Bl** | Blind — слепота |
| **Gl** | Gehörlos — глухота |
| **RF** | Befreiung Rundfunkbeitrag — освобождение от налога ТВ |

## Финансирование транспорта

| Услуга | Источник | Сумма |
|--------|----------|-------|
| Krankentransport | Krankenkasse | покрывает |
| Taxi для G/aG | Sozialamt | до €100/мес |
| Электроинвалидное кресло | Krankenkasse / Sozialamt | по нужде |
| Treppenlift | Pflegekasse | до €4 000 |
| Wohnungsumzug | Sozialamt | при оправдании |
