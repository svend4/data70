# 12 — INVEST-Zuschuss

**Программа:** INVEST — Zuschuss für Wagniskapital (BAFA, Министерство экономики)
**Сумма:** до 25% от ангельских инвестиций (макс. €100K)
**Условия:** для стартапов в Германии до 7 лет

## Что это

**INVEST-Zuschuss** — программа, по которой государство субсидирует **частных ангелов-инвесторов**, которые вкладывают в немецкие стартапы.

Это не прямое финансирование стартапа, а **налоговый вычет ангелу** = 25% обратно из казны.

## Как это работает

1. Ангел-инвестор находит стартап
2. Подписывают договор (минимум €10 000 ангел)
3. Вкладывает деньги в долю
4. Получает от государства 25% обратно (макс. €100K за один стартап)
5. Стартап получает **полную сумму** ангельских инвестиций

Эффект: для ангела это **снижение риска на 25%**, для стартапа — **более лёгкий поиск ангелов**.

## Условия

| Условие | Параметр |
|---------|----------|
| Возраст стартапа | до 7 лет |
| Регистрация | в Германии или ЕС |
| Сектор | инновационный (определяется BAFA) |
| Размер инвестиции | мин. €10K, макс. €500K |
| Доля ангела | 10-25% |
| Срок | вложение должно быть не менее 3 лет |

## Подходящие проекты

Все проекты автора подходят:
- SozialPlaner SaaS
- НейроОС
- CareMate Total
- SkyMediaHub Bavaria

## Стратегия использования

INVEST-Zuschuss — **акселератор поиска ангелов**, а не самостоятельный фандинг.

1. **Сначала бизнес-план + MVP**
2. **Потом поиск ангелов** через:
   - BAND (Business Angels Netzwerk Deutschland)
   - Reactor Saxony
   - LinkedIn outreach
3. **На встрече с ангелом** упомянуть INVEST как «бонус для вас»
4. **Оформление** — через юриста
