# 10 — GeoArb (географический арбитраж)

**Тема каталога:** #55
**Уровень готовности:** ★★★

## Что это

**Географический арбитраж** — стратегия, при которой человек/бизнес **зарабатывает в стране с высокими ценами и живёт в стране с низкими**. Ключевые параметры:

- Стоимость жизни (housing, food, transport)
- Налоги
- Здравоохранение
- Качество интернета и инфраструктуры
- Виза-режим
- Безопасность

## Применение

### Для фрилансера
- Контракты с DACH/US-клиентами (€80-150/час)
- Жизнь в Португалии / Грузии / Болгарии (€500-1500/мес)
- Прибыль: 70-90% от дохода

### Для микрофранчайзи
- Покупка микрофраншизы в DE → запуск в DE
- Управление из Чиангмая
- Лояльная экономика «Запад-Восток»

### Для СтартАпа
- Команда в Восточной Европе / Азии
- Маркетинг для DACH-рынка
- Юридическая база — Эстония (e-Residency)

## Технологии для GeoArb

| Категория | Инструменты |
|-----------|-------------|
| Юр. базы | Эстония e-Residency, ОАЭ Free Zone, Singapore |
| Налоговая оптимизация | Nomad Tax, Wise, Revolut |
| Жильё | Airbnb, Outsite, Selina |
| Сообщества | Nomad List, Remote Year |
| Здоровье | SafetyWing, World Nomad Insurance |

## Карта GeoArb (для DACH-фрилансера)

| Уровень комфорта | Стоимость/мес | Регион |
|------------------|---------------|--------|
| Минимум | €500-1000 | Грузия, Болгария, Албания |
| Средний | €1000-2000 | Португалия, Кипр, Чехия |
| Высокий | €2000-3500 | Испания (не Барселона), Греция, Италия |
| Премиум | €3500+ | Швейцария, Норвегия, Сингапур |

DE/AT/CH = премиум. Доход тут, расходы — где удобнее.

## Технология: Карта GeoArb

ПО, которое:
1. Показывает стоимость жизни в реальном времени
2. Сравнивает виза-режимы
3. Налоги для конкретного гражданства
4. Качество интернета (Speedtest интеграция)
5. Безопасность (UN data)
