# 00_index — Главная навигация монорепозитория

## Назначение

Эта папка служит центральным входом в новый монорепозиторий знаний. Она не заменяет старые файлы, а помогает разложить существующий архив по кластерам, уровням и проектным направлениям.

## Основные задачи

1. Дать единую карту всего архива.
2. Разделить темы на крупные кластеры.
3. Показать, какие файлы являются первоисточниками.
4. Упростить работу с проектами: книга, SaaS, грант, презентация, патент, сайт, бот.
5. Создать структуру, которую можно постепенно заполнять без потери старой информации.

## Правило развития

Новые материалы лучше добавлять не в корень, а в соответствующий кластер. Если тема относится к нескольким кластерам, главный файл кладётся в основной кластер, а в других создаются ссылки или короткие заметки.
