# 07 — RU↔DE перевод для русскоязычной аудитории

**Уровень готовности:** ★★★★

## Что это

Автоматический перевод между русским и немецким языками — критический компонент для:

- Книги «Persönliches Budget» (две версии)
- Telegram-бот SozialHelper
- AI-newsroom для русскоязычной диаспоры
- SozialPlaner SaaS (мультиязычный)

## Технологии

| Решение | Качество | Стоимость | Скорость |
|---------|----------|-----------|----------|
| **DeepL** | Лучшее для DE | $30/мес. | Быстро |
| **Google Translate** | Хорошее | Бесплатно/API | Быстро |
| **Claude / GPT** | Отлично с контекстом | API-биллинг | Средне |
| **Локальный M2M-100** | Хорошее | €0 (после установки) | Зависит от железа |
| **NLLB-200** (Meta) | Очень хорошее | €0 | Зависит |

## Особенности юридического перевода

| Сложность | Решение |
|-----------|---------|
| **Правовые термины** | Терминологический словарь (TM) |
| **Параграфы законов** | Не переводить, оставить «§ 29 SGB IX» |
| **Тон** | Формальный (для документов) / разговорный (для бота) |
| **Контекст** | Пройти через LLM с инструкцией |

## Применение в проектах

### Книга по PB (две версии)
- Базовая компиляция на одном языке (русском)
- Перевод на немецкий через DeepL + редактура
- Юр. термины проверяются вручную

### Telegram-бот
- Пользователь пишет на родном
- Бот отвечает на родном
- Документы генерируются на немецком (для подачи)

### AI-newsroom
- DE-источники → перевод на RU для диаспоры
- Сохранение оригинала для верификации
