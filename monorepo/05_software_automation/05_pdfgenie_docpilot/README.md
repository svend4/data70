# 05 — PDFGenie / DocPilot

**Тема каталога:** #61
**Приоритет:** Б (10)
**Уровень готовности:** ★★★

## Что это

**ИИ-агент, встроенный непосредственно в документ** (PDF, Word, Markdown). Не «отдельный чат», а агент, который **видит весь документ** и работает прямо в нём.

Аналогия: Microsoft Copilot, но для любых документов и форматов, локально.

## Применения

| Документ | Что делает агент |
|----------|------------------|
| PDF научной статьи | Суммаризация, проверка ссылок, рисование схем |
| Юридический Bescheid | Анализ оснований, формирование Widerspruch |
| Медицинская карта | Анализ трендов, рекомендации |
| Деловое письмо | Перевод, тональность, ответ |
| Книга | Резюме каждой главы |
| Контракт | Подсветка рисков |

## Что отличает от существующего

| Решение | Минус | Что отличает PDFGenie |
|---------|-------|----------------------|
| **Microsoft Copilot** | Только Office | Любой формат |
| **Adobe Acrobat AI** | Только Adobe | Open-source |
| **NotebookLM (Google)** | Только облако | Локально + приватно |
| **PDFGenie** | — | Универсал + локальный |

## Технологии

- **Локальный LLM** (Ollama, Llama 3.1)
- **Парсинг PDF** через PyMuPDF / pdfplumber
- **Word через docx**
- **OCR через Tesseract** для отсканированных документов
- **UI**: интеграция с Adobe-плагинами + standalone

## Применение в проектах автора

- **SozialPlaner** — анализ Bescheid от Sozialamt в PDF
- **CareMate** — анализ медицинских карт
- **Newsroom** — обработка PDF-исследований
- **Книга по SGB** — Copilot для редактирования и фактчекинга
