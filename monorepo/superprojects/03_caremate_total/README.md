# Суперпроект 3: CareMate Total

**Бюджет MVP:** €150-220K
**Срок MVP:** 5-6 мес
**Бизнес-модель:** B2B2C SaaS + Hardware Bundle
**MRR на 24 мес:** €40K
**Безубыточность:** ~24 мес

## Что это

Комплексная **ИИ-экосистема заботы** о человеке от рождения до старости:
- Цифровой Спутник (детство)
- Виртуальный учитель (подросток)
- Автоматизация быта (взрослый)
- CareMate Hub (старшее поколение)
- 105 сценариев заботы
