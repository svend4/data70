# 12 — Биомодель ИИ «Пчёлы и Муравьи» — патентная карточка

**Категория:** D (Архитектура)
**Новизна:** ★★★★
**Применимость:** ★★★
**Статус:** Архитектура (защита через публикацию + патент на конкретные нюансы)

## Суть

Двухуровневая мультиагентная архитектура ИИ:
- **Пчелиный слой** — тысячи микроагентов-исполнителей (узкая задача, дёшево, заменимо)
- **Муравьиный слой** — надзорные агенты-контролёры (распределение, проверка, оптимизация)

## Сравнение с MoE и MAS

| Решение | Структура | Особенность |
|---------|-----------|-------------|
| **MoE** (Mixture of Experts) | Эксперты внутри одной модели | Только инференс |
| **MAS** (Multi-Agent Systems) | Однотипные агенты | Без явной иерархии |
| **Hierarchical RL** | Уровни в reinforcement learning | Только обучение |
| **Bees & Ants** | Два слоя с биологической метафорой | Управленческая иерархия + дешёвые исполнители |

## Что заявляется

1. **Двухуровневая архитектура** с явным разделением: исполнители + контролёры
2. **Алгоритм назначения** агентов на задачи (на основе biological inspiration)
3. **Метод оценки качества** работы пчёл муравьями
4. **Стратегия эволюции** — пчёлы плохого качества заменяются

## Реализация

- **Пчёлы:** Llama 3.1 8B Q4 + узкие промпты, в LangGraph как параллельные узлы
- **Муравьи:** Claude / GPT-5 + системные промпты QA
- **Связь:** через `02_ai_agents/22_semantic_bus`

## Применение

- НейроОС — пчёлы для рутины, муравьи для качества
- SozialPlaner — пчёлы для генерации, муравьи для правовой проверки
- CareMate — пчёлы для сценариев, муравьи для безопасности
