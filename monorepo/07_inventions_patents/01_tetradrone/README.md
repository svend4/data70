# 01 — TetraDrone — патентная карточка

**Категория:** A (Дроны)
**Новизна:** ★★★★
**Применимость:** ★★★★★
**Статус:** **Подать патент**

## Суть изобретения

Корпус дрона, изготовляемый на стандартной упаковочной линии Tetra Pak. Многослойный материал (картон + полимер + фольга) даёт лёгкость, влагостойкость, биоразлагаемость основы.

**Ключевая инновация:** использование готовой производственной инфраструктуры Tetra Pak в Европе для массового производства корпусов дронов.

## Prior art (известное)

| Аналог | Особенность | Чем отличается |
|--------|-------------|----------------|
| **Sypaq PPDS** (AU) | Картонный одноразовый дрон | Нет интеграции с Tetra Pak линией |
| **Cardboard drones** (общая концепция) | Различные стартапы | Без многослойного материала |
| **DJI и др.** (классика) | Пластик/композит | Дорого, не биоразлагаемо |

## Что заявляется

1. **Метод производства** корпуса дрона на адаптированной линии Tetra Pak
2. **Развёртка корпуса** — конкретная геометрия из плоского листа
3. **Электронная упаковка** в многослойный материал (защита от влаги без дополнительных слоёв)
4. **Метод утилизации** биоразлагаемой основы

## Стратегия

1. **Поиск Patentanwalt** в Дрездене (специалист по UAV)
2. **Prior art search** через DPMA + EPO + Espacenet
3. **Adv заявка** в DPMA — €500-3K
4. **Параллельно** — связь с Tetra Pak Deutschland для лицензии на производственную линию
5. **Публикация** через 18 мес после подачи (стандарт)
