# 13 — Стандарты зарядки и безопасности роя

**Уровень готовности:** ★★★

## Что это

Унифицированные **стандарты зарядки и безопасности** для всех 15 роботов-зверей. Чтобы рой работал — нужны общие правила.

## Стандарт зарядки

| Аспект | Решение |
|--------|---------|
| Контактный разъём | USB-PD / Qi / магнитный (по типу робота) |
| Напряжение | 12 В (мелкие), 24 В (средние), 48 В (большие) |
| Зарядная станция | Многопортовая на 6-15 роботов |
| Управление зарядкой | MQTT — робот «просит» свободный порт |
| Приоритеты | Робот с критичной задачей → выше приоритет |
| Безопасность | Контроль температуры, отключение при отказе |

## Стандарт безопасности

### Физическая безопасность
- **Мягкие материалы** для всех контактных зон
- **Скорость** не более 1 м/с в присутствии человека
- **Сенсоры столкновения** на каждом роботе
- **Аварийный стоп** доступен из приложения / голосом «STOP»

### Кибербезопасность
- **Локальная коммуникация** — без облака
- **Зашифрованная связь** (TLS 1.3 для Matter, DTLS для DDS)
- **Аутентификация** каждого робота (X.509 сертификат)
- **Аудит** всех команд
- **Sandbox для команд от ИИ** — нельзя выполнять прямые системные команды

### Психологическая безопасность
- **Звуковые сигналы** перед движением (чтобы не пугать)
- **Визуальные индикаторы** статуса (LED-стрипы)
- **Ограничения зон** — робот не входит в спальню без разрешения
- **Право пользователя на «выключить»** — приоритет выше всех команд ИИ
