## Суть

Использование **языка Аймара** (древний южноамериканский) и **Forth** (компактный программный) как **промежуточных языков** для:
- Перевода между языками (Аймара как «нейтральный» язык)
- Описания вычислений (Forth как минимальный)

## Почему Аймара

Лингвист Иван Гусман де Рохас (Боливия) в 1980-х показал, что грамматика Аймара особенно близка к **алгоритмической логике**: каждое предложение содержит однозначную модальность (известное / неизвестное / неточное).

Это используется в SYSTRAN-проектах для машинного перевода.

## Почему Forth

Forth обладает уникальным свойством: **минимализм без потери выразительности**. Любую вычислительную задачу можно описать на Forth с минимальным числом слов.

## Концепция «двух промежуточных языков»

```
Естественный язык → Аймара → Forth → Машина
                  (логика)  (вычисление)
```

Этот двухпроходный путь даёт:
- Логическую чистоту (Аймара)
- Вычислительную эффективность (Forth)
- Композицию NLP + Embedded

## Применение

- Multilingual LLM с промежуточным Аймара
- Forth-runtime для NLP-задач на edge
- Образовательная система (Аймара + Forth как методология)
