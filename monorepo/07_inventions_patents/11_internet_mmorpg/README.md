# 11 — «Интернет как MMORPG» — концептуальная карточка

**Категория:** D (Концепции)
**Новизна:** ★★★
**Применимость:** ★★★
**Статус:** Концепция (защита авторским правом)

## Суть

Перепроектирование интернета так, чтобы **позитивные действия монетизировались**. Аналогия с MMORPG:
- Квесты — полезные задачи (волонтёрство, помощь, обучение)
- Гильдии — сообщества по интересам
- Пассивный доход — за полезные действия в прошлом
- «Гидроэлектростанция» — преобразование информационного потока в ценность

## Чем отличается

| Аналог | Особенность | Что добавляет идея |
|--------|-------------|--------------------|
| **Steemit / Hive** | Монетизация контента | Системность, не только контент |
| **BAT (Brave Browser)** | Микроплатежи за внимание | Активные действия, не пассивные |
| **Reddit Karma** | Бесплатная репутация | Конвертация в деньги |
| **Геймификация в приложениях** | В отдельных | Геймификация всего интернета |
