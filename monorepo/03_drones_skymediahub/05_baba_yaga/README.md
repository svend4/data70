# 05 — «Баба Яга» — тяжёлый гражданский дрон

**Тема каталога:** #68
**Уровень готовности:** ★★★

## Что это

**«Баба Яга»** — концепция тяжёлого гражданского мультикоптера для перевозки крупных грузов. Альтернатива малой авиации для доставки в труднодоступные районы.

Название отсылает к фольклорному образу «летящей избушки» — большой, странной, но эффективной.

## Спецификация (концептуальная)

| Параметр | Значение |
|----------|----------|
| Класс EU | C5/C6 (Specific category) |
| Взлётный вес | 25-150 кг |
| Полезная нагрузка | 10-50 кг |
| Дальность | 30-100 км |
| Скорость | 60-120 км/ч |
| Время полёта | 30-90 мин |
| Двигатели | 6-8 моторов с электроприводом |
| Энергия | LiPo / водородные топливные элементы |

## Применения (гражданские)

1. **Сельское хозяйство** — распыление на больших полях
2. **Доставка лекарств** в горные регионы (Альпы)
3. **Спасательные операции** — сброс грузов в зону катастроф
4. **Лесное хозяйство** — перевозка инвентаря
5. **Строительство в труднодоступных местах** — материалы на верх

## Регуляторика EU

C5/C6 требуют:
- Operational Authorization от EASA / Luftfahrtbundesamt
- Specific Operations Risk Assessment (SORA)
- Сертифицированный пилот
- Strognиe ограничения зон полёта

## Связь с темой #57 (WILOS)

«Баба Яга» — флагман проекта **«рой тяжёлых дронов»**:
- Несколько «Баба Яга» работают вместе для перевозки очень тяжёлых грузов
- Координация через WILOS (`07_inventions_patents/09_wilos_dsl`)
- Связь между дронами через mesh + оптоволокно (`04_optofiber_chain`)
