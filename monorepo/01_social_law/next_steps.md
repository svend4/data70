## Волна 2: SaaS (3-12 мес.)

**SozialPlaner** — веб-платформа для соцработников
- Каталог услуг (из `08_70_modules_catalog`)
- Генератор документов (из `02_widerspruch_templates`, `03_klage_sozialgericht`)
- Таймлайн дела (из `07_master_dossier`)
- Калькулятор бюджета (из `04_pflege_assistenz`)
- Стек: Next.js + Supabase + Claude API
- Цена: €29-99/мес.
- Финансирование: EXIST-Gründerstipendium €150K
- Срок MVP: 4-5 мес.

## Юридическая защита

- **Клиренс контента:** все шаблоны прошли через личное использование, риск претензий минимален
- **Лицензия:** CC BY-NC для свободного распространения, отдельные коммерческие условия для SaaS
- **GDPR:** персональные данные пользователей хранятся только локально (RAG offline-first)
