# 08 — Трёхуровневая модель управления

**Тема каталога:** #56 (хаб 11+ связей)
**Уровень готовности:** ★★★★

## Что это

Авторский **мета-паттерн «Три уровня всюду»** — деление любой системы на:

1. **Оперативный** — рутина, ежедневные действия
2. **Тактический** — координация, планирование
3. **Стратегический** — целеполагание, ценности

## Где применяется

| Сфера | Оперативный | Тактический | Стратегический |
|-------|-------------|-------------|----------------|
| **ИИ-архитектура** | Мелкие модели | MoE | Большие модели |
| **B2B MetaWorld** | Сделки | Партнёрства | Экосистемы |
| **Управление знаниями** | Заметки | Темы | Принципы |
| **Управление ИИ** | Микроагенты | Координаторы | Стратеги |
| **Финансы** | Расходы | Бюджеты | Инвестиции |

## Зачем это паттерн

1. **Ясность ролей** — каждый компонент знает своё место
2. **Дисциплина** — нельзя смешивать уровни
3. **Стандарт коммуникации** — слова «оперативный/тактический/стратегический» понятны всем
4. **Маркетинг** — автор использует это как фишку
