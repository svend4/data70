## Суть

**Промпт-фреймворк** для ИИ-агентов на основе геометрической метафоры:
- **Звезда** — центральный агент с 5-7 «лучами» (специализациями)
- **Сота** — окружение, контекст, ресурсы
- Каждый «луч» звезды отвечает за одну роль/задачу
- Сота — состояние мира, в котором действует звезда

## Метафора

```
        ▲
       /|\
      / | \
     /  |  \
   ◄────●────►   ← звезда-агент (центр)
     \  |  /
      \ | /
       \|/
        ▼
   ─ ─ ─ ─ ─ ─ ─
   |  | СОТА | |  ← окружение, ресурсы, контекст
   ─ ─ ─ ─ ─ ─ ─
```

## Зачем

Это **систематизация** того, как пишутся системные промпты:
1. Звезда задаёт «что я делаю» (роль, цель, ограничения)
2. Сота задаёт «где я нахожусь» (контекст, доступные данные, состояние)

Используется в архитектуре:
- Микроагентов (`02_ai_agents/01_micro_agents`)
- Биомодели Пчёлы и Муравьи (`12_bees_ants_arch`)
