# 18 — НейроОС Freemium бизнес-модель

**Тип:** Freemium SaaS + Marketplace
**MVP бюджет:** €100-150K
**Срок MVP:** 4-5 мес.
**Источник:** `part14_business_models.md` раздел 14.1

## Что это

**Внешняя бизнес-модель** для проекта НейроОС / Internet Function OS. Включает:

- Freemium SaaS (Free → Pro → Team → Enterprise)
- Маркетплейс агентов и Blueprint
- API для разработчиков
- Партнёрства с LLM-провайдерами

## Тарифы

| Тариф | Цена | Что входит |
|-------|------|-----------|
| **Free** | €0 | Локальная работа, 1 user, базовые агенты, Ollama |
| **Pro** | €9/мес | Облако-синхр., 5 устройств, продвинутые агенты, Claude API лимит |
| **Team** | €19/мес/чел | Коллаб., общая база знаний, админ-панель |
| **Enterprise** | от €499/мес | On-premise, SSO, SLA, custom intgr. |

## Дополнительные источники дохода

| Источник | Механизм | Прогноз |
|----------|----------|---------|
| Маркетплейс агентов | 30% комиссия | €5-50 за агента × 200 шт./мес |
| Маркетплейс YAML-пайплайнов | 25% комиссия | €10-100 за пайплайн × 100 шт./мес |
| API для разработчиков | Pay-per-call | €0.001-0.01 за запрос |
| Реферальная партнёрская программа | 10% от LLM-биллинга | долгосрочная |

## Целевая аудитория

- **Первичная (Free → Pro):** Разработчики, исследователи, power users
- **Вторичная (Team):** Малый бизнес 1-20 чел., юристы, аналитики
- **Третичная (Enterprise):** Корпорации, образовательные учреждения

## Путь к окупаемости

```
Месяц 1-6:    MVP, бета, 500 free
Месяц 7-12:   Запуск Pro, 2000 free + 200 Pro = €1.8K MRR
Месяц 13-18:  Team + маркетплейс, 5000 free + 800 Pro + 50 Team = €12K MRR
Месяц 19-24:  Enterprise, первые B2B = €35-50K MRR
Месяц 24:     Безубыточность при ~€40K MRR
```

## KPI

- MAU (целевая 5000 на 24 мес.)
- Free → Pro конверсия (цель: 8-12%)
- Retention 30-й день (цель: >40%)
- Кол-во агентов в маркетплейсе (цель: 200+ к году)
