## Трёхуровневая система оплаты (#82)

| Уровень | Что | Аудитория |
|---------|-----|-----------|
| **1. Токены** | 1 токен = 1 цент за конкретную статью/действие | Любой читатель |
| **2. Криптовалюты** | USDT, BTC, ETH для пакетов токенов | Технические пользователи |
| **3. Классические системы** | Подписки, корпоративный доступ | Институции |

## Стратегия для автора

**Не делать ICO в первую очередь.** Начать с **off-chain учёта** (классическая подписка), потом постепенно интегрировать токены как **дополнительный** способ оплаты.

Этапы:
1. **Mes 0-12:** Stripe + классические подписки
2. **Mes 12-24:** Опциональная оплата USDT/BTC через NowPayments
3. **Mes 24+:** Свой утилитарный токен (через Solana или Polygon)
4. **Когда регуляторно безопасно:** stablecoin-расчёты
