# 06 — Kickstarter

**Тип:** Crowdfunding (предзаказы, без equity)
**Сумма:** €30K-100K (типичный успех в hardware)
**Срок:** 30-60 дней кампании

## Что это

Кикстартер — самый популярный crowdfunding-сайт мира. Идеально подходит для hardware-проектов с **визуальным wow-фактором**.

## Подходящие проекты автора

### 1. TetraDrone Edu Kit
- 5 дронов из Tetra Pak + ПО + инструкция
- Цена: €99-199 (early bird), €149-299 (regular)
- Целевая сумма: €30K
- Аудитория: учителя STEM, родители, школы

### 2. CareMate Robot Zoo Kit
- 3 базовых робота-зверя (Снегирь, Бобр, Стриж)
- Цена: €299-599
- Целевая сумма: €100K
- Аудитория: техно-энтузиасты, ранние usability-пользователи

### 3. Forth-Рой Educational Pack
- 5 малых дронов с Forth-интеллектом + WILOS-обучалка
- Цена: €499-999
- Целевая сумма: €50K
- Аудитория: университеты, образовательные центры

## Структура успешной кампании

1. **Видео 2-3 мин.** — ключ к успеху
2. **Реалистичные обещания** — сроки, спецификации
3. **Stretch goals** — мотивация бэкеров
4. **Прозрачность** — еженедельные обновления
5. **Сообщество** — Discord/Telegram активный канал

## Затраты

| Статья | €/проект |
|--------|----------|
| Видео-продакшн | 2-5K |
| Маркетинг (email, ads) | 5-10K |
| Прототипы для съёмки | 3-10K |
| Платформенная комиссия | 5% от сбора |
| Платёжные комиссии | 3-5% |
| **Итого** | **15-30K** |

При сборе €100K — чистый доход €60-70K (после производства первой партии).

## Альтернативы

| Платформа | Особенность |
|-----------|------------|
| **Indiegogo** | Гибче условия, перпетуальная кампания |
| **Startnext** (DE) | Немецкая аудитория, лояльная |
| **Crowdfunder** | UK-фокус |
| **Kickstarter** | Самая большая аудитория |

## Стратегия

1. **Сначала собрать аудиторию** — 5K+ подписчиков перед запуском
2. **Закрытая бета** — 50-100 ранних пользователей
3. **Запуск в первую неделю** — 30-50% сбора
4. **Stretch goals** — расширение
