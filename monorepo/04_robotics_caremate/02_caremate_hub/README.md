**Тема каталога:** #62
**Приоритет:** Б (9)
**Уровень готовности:** ★★★★

## Что это

**CareMate Hub** — система ИИ-поддержки для людей с инвалидностью и пожилых, объединяющая мониторинг здоровья, безопасность, коммуникацию и автоматизацию быта.

Ядро суперпроекта **CareMate Total** (`superprojects/03_caremate_total/`).

## Архитектура — 6 модулей

| Модуль | Что делает |
|--------|-----------|
| **1. Мониторинг здоровья** | Wearables, биометрия, тренды, оповещения |
| **2. Безопасность** | Падение, дым, газ, открытые двери, кнопка SOS |
| **3. Комфорт** | Климат, свет, звук, голос-команды |
| **4. Когнитивная поддержка** | Напоминания, разговоры, тренировки памяти |
| **5. Связь с персоналом / семьёй** | Видеозвонок, уведомления, отчёт о состоянии |
| **6. Аналитика** | Тренды для врача, отчёты для Pflegekasse |

## Связь с архитектурой У-Синь

См. `03_wuxing_architecture`. CareMate спроектирован по 5 стихиям китайской философии:
- **Вода** — ввод (мониторинг, сенсоры)
- **Дерево** — анализ (ИИ, классификация)
- **Огонь** — реакция (немедленное действие)
- **Земля** — память (история, тренды)
- **Металл** — контроль (правила, безопасность)

## Сценарий применения (пример)

> Старик в квартире падает. Сенсор движения + камера фиксируют падение. CareMate Hub:
> 1. Голосовой опрос: «Вы в порядке?»
> 2. Если нет ответа 30 сек — звонок в семью
> 3. Параллельно — открытие входной двери для скорой
> 4. Запись инцидента в Pflegekasse-отчёт
