# 05 — Водный дрон на гидрофойлах — патентная карточка

**Категория:** A (Морские USV)
**Новизна:** ★★★
**Применимость:** ★★★
**Статус:** Концепт

## Суть изобретения

**Водный БПЛА** дельтавидной формы на гидрофойлах. Подводные крылья поднимают корпус над водой при V > 12 км/ч → значительное снижение сопротивления и экономия энергии.

## Prior art

| Аналог | Особенность | Чем отличается |
|--------|-------------|----------------|
| **Saildrone** (US) | Парусный USV | Гидрофойлы вместо паруса |
| **Liquid Robotics Wave Glider** | На волнах | Активное движение |
| **Военные USV** | Дорогие | Гражданская ниша |
| **Гражданские катамараны USV** | На водоизмещении | Гидрофойл = быстрее |

## Что заявляется

1. **Геометрия корпуса** (дельтавидная)
2. **Конструкция гидрофойлов** для USV
3. **Метод управления** (стабилизация при подъёме)
4. **Гибридная энергетика** (батарея + солнечная)

## Применение

- Морской мониторинг (камеры, гидрофоны)
- Поисково-спасательные операции
- Биомониторинг (китообразные, кораллы)
- Доставка между судами
- Военная dual-use (разведка)

## Стратегия

1. Концептуальный дизайн с инженером гидродинамики
2. Контакт с Saildrone / Liquid Robotics — бенчмарк
3. POC с уменьшенной моделью
4. Подача при подтверждении уникальности
