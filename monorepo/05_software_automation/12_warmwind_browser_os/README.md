# 12 — Облачная ОС через браузер

**Темы каталога:** #47, #77
**Уровень готовности:** ★★★

## Что это

**Облачная ОС**, работающая полностью **в браузере**. На устройстве — только браузер (или минимальный Linux). Файлы, приложения, база знаний — всё в облаке.

Эталоны:
- ChromeOS (Google)
- Warmwind OS web-interface
- Stadia (gaming, закрыто)
- Heroku Spaces
- StackBlitz / GitHub Codespaces

## Зачем это нужно

1. **Старые устройства** — браузер запускает любой ПК старше 10 лет
2. **Школы и развивающиеся страны** — дешёвые «тонкие клиенты»
3. **Безопасность** — нет местных данных, всё в облаке
4. **Кросс-платформа** — Windows / Linux / Mac / Chromebook
5. **Простота настройки** — открыть URL и работать

## Архитектура

```
Браузер (любой современный)
   ↓ HTTPS / WebSocket
┌────────────────────────────┐
│  Cloud OS Frontend (PWA)   │
│  - File manager            │
│  - Terminal                │
│  - Editor                  │
│  - Browser-in-browser      │
└─────────┬──────────────────┘
          ↓
┌────────────────────────────┐
│  Backend (Hetzner / VPS)   │
│  - User space (per user)   │
│  - Файлы (NextCloud / S3)  │
│  - Apps (через скрипты)    │
└────────────────────────────┘
```
