## Что это

**Warmwind OS** — операционная система для ИИ-агентов с упором на RPA (Robotic Process Automation). Агенты работают с GUI приложений как человек: видят экран, кликают, печатают.

Используется как ориентир для архитектуры НейроОС.

## Чем интересна

1. **Все приложения как объект** — не нужны API
2. **Универсальность** — работает с любым ПО
3. **Запись макросов** — пользователь показывает, агент повторяет
4. **Vision-LLM на десктопе** — экран как вход модели
