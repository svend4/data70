# 08 — DiGA-сертификация

**Уровень готовности:** ★★★

## Что это

**DiGA** (Digitale Gesundheitsanwendungen) — цифровые медицинские приложения, **сертифицированные BfArM** (Federal Institute for Drugs and Medical Devices).

Преимущества DiGA:
- **Финансирование Krankenkasse** — бесплатно для пациентов с рецептом
- **Признание в системе** — врачи могут прописывать
- **Регулярный доход** — Pauschalpreis с Krankenkasse

## Процесс DiGA

| Этап | Содержание | Срок |
|------|-----------|------|
| 1. Подготовка | Демо, безопасность, GDPR | 2-6 мес |
| 2. Подача в BfArM | Полный пакет документов | 1 мес |
| 3. Fast-track | 3 месяца на проверку | 3 мес |
| 4. Если одобрено | Listing in DiGA-Verzeichnis | — |
| 5. Цена | Переговоры с GKV-Spitzenverband | 1-3 мес |
| 6. Использование | Врачи прописывают | — |

## Стоимость

| Этап | Стоимость |
|------|-----------|
| Подготовка (юр., тех., клин.) | €20-40K |
| BfArM Fast-Track Gebühren | €15K |
| Аудит безопасности (BSI) | €20-50K |
| **Итого** | **€55-105K** |

## Что подходит для DiGA

| Применение DiGA | Пример |
|-----------------|--------|
| Депрессия | Selfapy, deprexis |
| Беспокойство | HelloBetter |
| Пациентское ведение | Cara Care |
| **CareMate Family Plus** | потенциальный кандидат |

## Связь с CareMate

CareMate Family Plus может стать DiGA, если:
- Терапевтическая ценность доказана (RCT)
- Безопасность подтверждена (BSI)
- GDPR-compliance проверен

При успехе:
- Krankenkasse платит ~€500-1500/3 мес (depending on model)
- Большая часть рынка стает доступной

## Стратегия

1. **MVP без DiGA** (как обычное приложение) — 6-12 мес
2. **Сбор данных** для клинического исследования — 6-12 мес
3. **Подготовка DiGA-заявки** — 3 мес
4. **Подача в BfArM** — Fast-Track 3 мес
5. **При успехе** — масштабирование через рецепты
