# 08 — Контент с дронов: обработка медиа

**Уровень готовности:** ★★★

## Что это

Технологии **обработки видео и фото с дронов** для AI-newsroom:

- Извлечение событий
- Описание сцены
- Распознавание объектов
- Генерация дескрипций
- Сшивка панорам (для составной 4K-камеры)

## Pipeline

```
Видео с дрона (4K)
   ↓ декодирование
Кадры (10 fps достаточно)
   ↓ Vision-LLM (CLIP / GPT-4V / Claude Vision)
Описания событий + temporal links
   ↓ агрегация
Статья / репортаж
```

## Технологии

| Задача | Технология |
|--------|-----------|
| **Object detection** | YOLOv8 / YOLOv9 |
| **Scene understanding** | CLIP, BLIP-2 |
| **Описание** | GPT-4V, Claude Vision |
| **Tracking** | DeepSORT, ByteTrack |
| **Сшивка** | OpenCV stitching |
| **NDVI обработка** | Custom GDAL |
