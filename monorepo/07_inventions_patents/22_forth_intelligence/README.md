## Суть

**Бортовой ИИ на стековом языке Forth**, занимающий **<4 КБ RAM** и работающий на любом микроконтроллере (ESP32, STM32, RP2040).

Сочетание:
- **eForth** — компактный Forth-интерпретатор
- **1-bit модели** (BitNet) — квантизованные нейросети
- **WILOS** runtime — исполняет язык роя

## Прот

Forth — стековый язык программирования 1970-х, известный своей **компактностью**:
- Базовое ядро — 2-4 КБ
- Запускается на 8-битных MCU
- Используется в космонавтике, embedded

## Что отличает

| Аналог | Размер | Чем отличается |
|--------|--------|----------------|
| **TensorFlow Lite Micro** | 16-100 КБ | Forth — компактнее |
| **Embedded ML (Edge Impulse)** | подобный | Закрытый, не open-source |
| **Tinygrad** | 5-10 КБ | Близко |
| **Forth-интеллект** | <4 КБ | Минимум возможного + DSL |

## Что заявляется

1. **Архитектура** Forth + BitNet runtime
2. **Метод дистилляции** больших моделей в Forth-выполняемые
3. **Расширение Forth** для нейровычислений (новые слова: `convolve`, `attend`, `softmax`)
4. **Mapping WILOS → Forth байткод**

## Применение

- **Каждый дрон в Forth-Рое** — бортовой ИИ <4 КБ
- **CareMate IoT-сенсоры** — простой ИИ для распознавания (падение, дым)
- **Образовательные роботы** — простота языка для обучения

## Стратегия

1. **POC eForth + BitNet** на ESP32-S3 (8 МБ RAM)
2. **Дистилляция** простой модели в Forth-выполняемую
3. **Бенчмарк** на простой задаче (классификация изображения 32×32)
4. **Эссе** «Forth + 1-bit LLM = ultra-tiny edge AI» на Hacker News
5. **Подача DPMA** на конкретные нюансы
