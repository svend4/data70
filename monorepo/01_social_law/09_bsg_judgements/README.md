# 09 — Решения Bundessozialgericht (BSG)

**Кластер:** 01_social_law
**Уровень готовности:** ★★★

## Что это

Подборка ключевых решений Bundessozialgericht (Высший социальный суд Германии) по темам, релевантным делам автора. Используется как **прецедентная база** для аргументации Widerspruch и Klage.

## Тематические блоки

| Тема | Кол-во решений | Применение |
|------|----------------|-----------|
| Persönliches Budget | 6+ | `01_persoenliches_budget` |
| § 14 SGB IX (переадресация) | 4+ | `06_kostenschieberei` |
| Pflegegrad / переоценка | 5+ | `04_pflege_assistenz` |
| Doppelausstattung (§ 84 SGB IX) | 2+ | `04_pflege_assistenz` |
| Schonvermögen (§ 90 SGB XII) | 3+ | `15_kindergeld_ao` |
| Ассистенция в Werkstatt | 2+ | `12_assistive_services_50` |
| Wohnungsräumung / Umzug | 2+ | `15_kindergeld_ao` |

## Структура карточки прецедента

Для каждого решения заведена карточка:

- **Aktenzeichen** (B 8 SO 5/22 R и т.п.)
- **Дата решения**
- **Стороны** (краткое описание)
- **Основной вопрос**
- **Решение** (тезисно)
- **Главные правовые тезисы** (Leitsätze)
- **Ссылка на полный текст** (juris.de или sozialgerichtsbarkeit.de)
- **Применение в делах автора** (где использовать)

## Применение

1. **При составлении Widerspruch** — цитата из BSG как «тяжёлый» аргумент
2. **При Klage** — Leitsatz BSG как обоснование
3. **В книге «Persönliches Budget»** — раздел «Что говорит BSG»
4. **В SaaS SozialPlaner** — модуль «Прецеденты по теме»

## Связи
