# 05 — AAL Programme (Active and Assisted Living)

**Программа:** AAL — Joint Programme by Active and Assisted Living
**Сумма:** до €500 000 (на консорциум)
**Условия:** ИКТ для пожилых людей, минимум 3 партнёра из 3 стран

## Что это

AAL Programme — европейская программа финансирования **технологий для активной и ассистированной жизни пожилых**. Особенно ценная для CareMate.

Состоит из:
- **Калл AAL** (ежегодный) — €25-30M распределяется на консорциумы
- **Active Assisted Living Forum** — конференция
- **Marketplace** — каталог решений

## Что финансирует

| Тема | Примеры |
|------|---------|
| Health monitoring | Wearables, мониторинг здоровья |
| Safety | Падение, дым, газ |
| Mobility | Помощники, навигация |
| Communication | Связь с семьёй и врачами |
| Independence | Поддержка самостоятельности |
| Cognitive support | Тренировки, напоминания |
| Social inclusion | MMORPG, чат, сообщества |

## CareMate Total — идеальный кандидат

Все 6 модулей CareMate Hub соответствуют темам AAL:

| Модуль CareMate | Тема AAL |
|-----------------|----------|
| Мониторинг здоровья | Health monitoring |
| Безопасность | Safety |
| Комфорт | Independence |
| Когнитивная поддержка | Cognitive support |
| Связь | Communication |
| Аналитика | Health monitoring (тренды) |

## Условия консорциума

Минимум 3 партнёра из 3 разных стран EU:
- 1 — стартап (=автор)
- 1 — академический партнёр (TU Dresden, Universität Wien, и т.д.)
- 1 — конечный пользователь (Caritas, Diakonie, дом престарелых)

## Структура заявки

1. **Цель** — какую боль решаем
2. **Состояние рынка** — конкуренты, ниша
3. **Инновация** — что нового даёт
4. **Технологический подход** — стек, архитектура
5. **Консорциум** — кто за что отвечает
6. **План работ** — 24-36 месяцев
7. **Бюджет** — €500K на консорциум
8. **Воздействие** — кому помогает, сколько людей

## Сроки

- **Объявление кalla** — обычно январь-февраль
- **Дедлайн заявок** — май
- **Решение** — октябрь
- **Старт проекта** — январь следующего года

1. Связаться с **TU Dresden — CeTI** (партнёр по технологии)

4. Изучить актуальный AAL Call
5. Подготовить заявку (8-12 нед.)
