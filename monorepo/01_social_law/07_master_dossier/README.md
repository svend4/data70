# 07 — Master Dossier v1.0

**Кластер:** 01_social_law
**Уровень готовности:** ★★★★★

## Что это

Master Dossier — заполняемый шаблон **полного личного дела** по соцсистеме Германии. Объединяет в одном документе всё, что нужно для ведения дел во всех ведомствах: реквизиты, ходатайства, переписку, сроки, расчёты.

## Структура шаблона

1. **Личные данные** — имя, адрес, страховой номер, GdB, Pflegegrad
2. **Реквизиты ведомств** — Sozialamt, KSV, Pflegekasse, Krankenkasse, Wohnungsamt, Versorgungsamt
3. **Реестр заявлений** — таблица Antrag/Bescheid/Widerspruch/Klage с датами и Aktenzeichen
4. **Реестр документов** — справки, заключения врачей, экспертизы (с отметкой «оригинал/копия»)
5. **Финансовый блок** — все выплаты, доходы, имущество (для проверки Schonvermögen)
6. **Текущие сроки** — что когда истекает
7. **Стратегия** — какой следующий шаг
8. **История корреспонденции** — все письма с/в ведомства

## Зачем это нужно

- **Личное:** один источник правды; легко передать представителю/юристу
- **Юридическое:** доказательная база при Klage и Aufsichtsbeschwerde
- **Коммерческое:** «обёртка» SaaS SozialPlaner — пользователь видит свой Master Dossier как dashboard

## Принципы

- **Заполняемый Markdown / YAML / JSON** — для машинной обработки в SaaS
- **Версионирование** — каждое изменение фиксируется (datetime)
- **Шифрование** — содержит чувствительные данные

## Структура папки (предложение)

```
master_dossier/
├── personal.yaml        # личные данные
├── institutions.yaml    # реквизиты ведомств
├── applications.yaml    # реестр заявлений
├── documents/           # PDF документов
├── correspondence/      # письма (Markdown)
├── timeline.yaml        # сроки и ключевые даты
└── strategy.md          # текущая стратегия
```
