# 12 — AAL Programme и e-Health

**Уровень готовности:** ★★★★

## Что это

**AAL** (Active and Assisted Living) Programme — главный европейский фонд для **технологий поддержки пожилых и инвалидов**. См. также `06_business_funding/05_aal_programme`.

## e-Health в Германии

| Программа | Описание |
|-----------|----------|
| **DiGA** | Цифровые медприложения (см. `08_diga_certification`) |
| **Telematikinfrastruktur** | Государственная сеть здравоохранения |
| **ePA** (elektronische Patientenakte) | Электронная карта пациента |
| **eRezept** | Электронный рецепт |
| **TI-Messenger** | Защищённый мессенджер для здравоохранения |

## Связь AAL + DiGA + CareMate

```
CareMate концепция
   ├── AAL Programme финансирование (R&D)
   ├── DiGA сертификация (продукт)
   └── ePA интеграция (данные)
```

## Перспективы

CareMate Total — **естественный кандидат** для AAL + DiGA:
- AAL — €500K на исследования
- DiGA — путь к €1500/3 мес от Krankenkasse за пациента
- ePA-интеграция — для обогащения сценариев

## Связи

2. Параллельно — DiGA-подготовка
3. ePA-интеграция через TI-Messenger
