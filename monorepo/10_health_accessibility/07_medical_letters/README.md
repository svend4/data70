# 07 — Медицинские письма

**Источник:** разговор #1 — 1365 сообщений, 2.6 МБ (один из самых проработанных)
**Уровень готовности:** ★★★★★

## Что это

Дополнение к `01_social_law/11_medical_letters`. Здесь — **медицинская сторона** работы с письмами:

- Как формулировать врачам
- Какие термины использовать
- Какие документы нужны
- Перевод между «врачебным» и «бюрократическим» языками

## Структура медицинского письма

| Часть | Содержание |
|-------|-----------|
| Заголовок | «Bescheinigung über Pflegebedarf» |
| Реквизиты | Врач, клиника, дата |
| Цель | «Zur Vorlage beim Sozialamt» |
| Анамнез | Диагнозы, история |
| Текущее состояние | Что не может пациент |
| Потребности | Часы, виды помощи |
| Рекомендации | Hilfsmittel, услуги |
| Прогноз | Стабилизация / ухудшение / улучшение |
| Подпись | Stempel врача |

## Ключевые формулировки

### Что хорошо для бюрократии
- «Patient ist auf Hilfe angewiesen»
- «Unfähigkeit zur selbstständigen Verrichtung»
- «Dauerhafte Pflegebedürftigkeit»
- «Schwerbehinderung mit GdB 70»

### Что плохо
- «Manchmal Probleme mit...» (слишком мягко)
- «Wenn nötig...» (условно)
- «Möglicherweise hilfreich...» (без обязательности)

## Применение

- Перед каждым Antrag PB
- При оспаривании MDK-Gutachten
- Для Klage в Sozialgericht
