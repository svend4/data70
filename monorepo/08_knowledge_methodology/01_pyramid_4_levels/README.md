# 01 — Четырёхуровневая пирамида знаний

**Тема каталога:** #41 (хаб 12+ связей)
**Уровень готовности:** ★★★★★

## Что это

**Авторская методология** управления информацией с 4 уровнями:

```
СТРАТЕГИЧЕСКИЙ — общая картина, цели, принципы
       ↓ ↑
ТАКТИЧЕСКИЙ    — годовое/квартальное планирование
       ↓ ↑
ОПЕРАТИВНЫЙ    — недельные/ежедневные задачи
       ↓ ↑
ПРАКТИЧЕСКИЙ   — конкретные документы, переписка, заметки
```

## Алгоритмы движения

### Снизу вверх: хаос → структура → энциклопедия
1. Сырые заметки попадают в **Практический** уровень
2. Систематизируются в **Оперативный** (по теме, по проекту)
3. Обобщаются в **Тактический** (планы, отчёты)
4. Выводятся принципы для **Стратегического**

### Сверху вниз: цель → декомпозиция → действие
1. **Стратегический** уровень задаёт цели
2. Разбиваются на **Тактические** планы
3. Декомпозируются в **Оперативные** задачи
4. Реализуются в **Практическом** уровне

## Связь с темой #56 (трёхуровневое управление)

Пирамида (4 уровня) — расширение `08_three_level_management` (3 уровня) добавлением **«Практического»**: реальные документы, не абстрактные операции.

## Применение

| Контекст | Где работает |
|----------|--------------|
| **Социальные дела автора** | Стратегия = «получить PB» → Тактика = «параллельная подача» → Оперативная = «Antrag» → Практическое = «реальный документ» |
| **Книга** | Стратегия = «послание» → Тактика = «главы» → Оперативное = «параграфы» → Практическое = «конкретные предложения» |
| **Продукт** | Стратегия = «миссия» → Тактика = «features» → Оперативное = «спринты» → Практическое = «код» |
