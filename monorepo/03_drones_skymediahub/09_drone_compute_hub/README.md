# 09 — DroneCompute Hub

**Тема каталога:** #64
**Уровень готовности:** ★★★

## Что это

**DroneCompute Hub** — наземный куб-хаб (cube-hub), служащий:
- Точкой посадки/старта дронов
- Edge-вычислительным центром (обработка видео, ИИ-инференс)
- Зарядной станцией
- Узлом оптоволоконной магистрали (`04_optofiber_chain`)

## Архитектура

```
                  ┌─ Антенна Wi-Fi/LoRa
                  │
   ┌──────────────┴──────────┐
   │   DroneCompute Hub      │
   │   ────────────────────  │
   │   - Зарядные станции    │  ← дроны садятся
   │   - Edge GPU/NPU        │  ← обработка видео
   │   - Хранилище SSD       │
   │   - UPS / солнечная     │
   │   - Оптоволоконный порт │  ← магистраль
   │   - Контейнер 1×1×2 м   │
   └─────────────────────────┘
```

## Спецификация (концепт)

| Компонент | Реализация |
|-----------|-----------|
| Корпус | Стандартный 19" rack контейнер на колёсах |
| Energy | Солнечные панели на крыше + LiPo буфер |
| Compute | NVIDIA Jetson AGX / Orin / x86 mini-PC |
| Storage | SSD NVMe 4-16 ТБ |
| Сеть | Optical fiber + 5G + Wi-Fi 6 + LoRa |
| Зарядка | Контактная (как у Tesla Wall Connector) |
| ИИ | Локальный Ollama + специализированные модели |

## Применения

1. **Сельхоз-кооператив** — куб-хаб посреди поля, агро-дроны летают и заряжаются
2. **Стройплощадка** — мониторинг с воздуха в реальном времени
3. **Морская платформа** — мониторинг судоходства
4. **Военная база** — разведка периметра
5. **Спасательные операции** — мобильный hub на грузовике

## Edge AI на куб-хабе

- Распознавание объектов в видеопотоке
- Сшивка видео с массива дронов (для составной 4K-камеры)
- ML-классификация (типы посевов, повреждения, угрозы)
- Локальная база векторов для RAG-поиска по архиву видео
