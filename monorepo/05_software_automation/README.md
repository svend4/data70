## О чём этот кластер

Кластер собирает все материалы по **Internet Function OS (НейроОС)**, no-code/low-code, автоматизации (n8n, Make.com), RPA, виртуальным офисам и архитектуре нового интернета.

Главный актив — **Internet Function OS**: полное ТЗ + MVP Backlog + Function Registry. Это база для суперпроекта НейроОС.
