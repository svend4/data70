# FloraShop — Магазин цветов

Веб-сайт магазина цветов с каталогом товаров, корзиной и формой заказа.

## Возможности

- Каталог цветов с фильтрацией по категориям (букеты, розы, свадебные, комнатные)
- Корзина покупок с изменением количества товаров
- Форма заказа с контактными данными
- Адаптивный дизайн для мобильных устройств
- Уведомления при добавлении товаров

## Структура проекта

```
├── index.html       # Главная страница
├── css/
│   └── style.css    # Стили
├── js/
│   └── app.js       # Логика приложения
└── README.md
```

## Запуск

Откройте `index.html` в браузере. Сервер не требуется — это статический сайт.
