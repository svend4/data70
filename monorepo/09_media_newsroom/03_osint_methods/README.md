# 03 — OSINT методология

**Уровень готовности:** ★★★★

## Что это

**OSINT** (Open Source Intelligence) — сбор и анализ информации из открытых источников. Авторская методология включает:

- Систематический поиск
- Верификация источников
- Triangulation (подтверждение через 2-3 источника)
- Документирование цепочки доказательств

## Источники для разных тем

### Социальное право (DE)
- BSG, BVerfG (решения судов)
- BMAS (Министерство труда)
- Sozialgerichtsbarkeit.de
- Anwalt.de, kanzlei.de
- VdK, Sozialverbände

### ИИ и технологии
- arXiv (статьи)
- Hacker News, Reddit r/MachineLearning
- Twitter/X (исследователи)
- GitHub (open-source проекты)

### Дроны и БПЛА
- EASA (регуляция)
- DLR публикации
- USPTO патенты
- Defense World, Aviation Week

### Бизнес и финансы
- Crunchbase
- LinkedIn (компании)
- BeFa (Bundesanzeiger финансовые отчёты)
- Branchenverbände

## Этический OSINT

| Принцип | Описание |
|---------|----------|
| **Только открытые источники** | Не взламывать пароли |
| **Не нарушать GDPR** | Не агрегировать персональные данные |
| **Проверка перед публикацией** | Triangulation |
| **Атрибуция** | Указывать источники |
| **Защита персон** | Анонимизировать при необходимости |
