# 02 — SmartOfficeProto

**Уровень готовности:** ★★★★★ (спецификация + бюджет)

## Что это

**Виртуальный офис** в виде **трёхпанельного интерфейса**:

```
┌──────────────────────────────────────────────┐
│ [ Логотип / меню ]              [ Юзер ↓ ]  │
├──────────────┬───────────────┬──────────────┤
│              │               │              │
│ РАБОЧАЯ ЗОНА │  ИИ-ЧАТ       │  KANBAN      │
│              │               │  / TODO      │
│ (Документ,   │  Ассистент с  │              │
│  таблица,    │  контекстом   │  Доска задач │
│  код, изобр.)│  всего экрана │  с превью    │
│              │               │              │
│              │               │              │
└──────────────┴───────────────┴──────────────┘
```

## Целевые роли

1. **Фрилансер** — рабочая среда + ИИ-помощник + Kanban проектов
2. **Студент** — учебная зона + ИИ-наставник + список курсовых
3. **Команда** — общая рабочая зона + общий чат + общая Kanban
4. **Человек с инвалидностью** — упрощённый интерфейс + голос + крупные элементы

## Стек

- **React + TypeScript** — фронтенд
- **Tauri** — десктопное приложение (легче Electron в 10x)
- **PWA** — для мобильных
- **Ollama** — локальный ИИ
- **WebSocket** — real-time обновление

## Что отличает

| Решение | Минус | Что отличает SmartOffice |
|---------|-------|-------------------------|
| Notion | Нет ИИ-чата с контекстом | Чат видит экран |
| ChatGPT | Нет рабочей зоны | Документ + чат вместе |
| Cursor | Только код | Любой контент |
| **SmartOfficeProto** | — | Универсал + локальный ИИ |
