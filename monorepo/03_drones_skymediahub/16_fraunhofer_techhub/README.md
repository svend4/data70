## Что это

Расширенная контактная база ключевых германских исследовательских и инкубационных организаций для проектов SkyMediaHub.

## Fraunhofer-Gesellschaft

Сеть из 76 институтов, крупнейшая прикладная R&D-организация в Европе.

| Институт | Локация | Тема для SkyMediaHub |
|----------|---------|---------------------|
| **IPMS** (Photonic Microsystems) | Dresden | Оптика, чипы для камер |
| **FOKUS** (Open Communication) | Berlin | Сети, оптоволокно |
| **EMI** (Ernst Mach) | Freiburg | Аэродинамика, материалы |
| **HHI** (Heinrich Hertz) | Berlin | Видеообработка |
| **IIS** (Integrated Circuits) | Erlangen | Чипы для AI/edge |
| **IZM** (Reliability/Microintegration) | Berlin | Надёжность электроники |

## TechHUB SVI Sachsen

**Smart Systems Hub** — один из 12 Digital Hubs Initiative Германии. Специализация: **умные системы, IoT, AI, robotics**.

- **Локация:** Dresden
- **Партнёры:** Fraunhofer, TU Dresden, Telekom, SAP
- **Программы:** инкубация, кофинансирование, доступ к лабораториям
- **Контакт:** smart-systems-hub.de
