# 13 — Цифровой Спутник Жизни — патентная карточка

**Темы каталога:** #27, #28
**Категория:** D (Концепции)
**Новизна:** ★★★★
**Применимость:** ★★
**Статус:** Концепция

## Суть

**Пожизненный ИИ-компаньон** — дрон/робот-Тамагочи, сопровождающий человека **с детства до старости**. Совмещает:

- Учитель (виртуальная школа)
- Партнёр по работе (виртуальная работа)
- Защитник (мониторинг здоровья и безопасности)
- Совесть (нравственный модуль)
- Архив (вся жизнь в памяти)

## Прот: Лампа Аладдина / Ангел-хранитель

Метафора:
- **Ангел-хранитель** — поддержка, советы
- **Искуситель** — оппонент, проверка решений
- **Крёстный** — заботится о пути

Это не «помощник», а **полноценная личность** в виде ИИ.

## Что отличает

| Аналог | Особенность | Чем отличается |
|--------|-------------|----------------|
| **Siri / Alexa** | Помощники | Не персональные на годы |
| **Replika** | Чатбот-друг | Не пожизненный, без нравственности |
| **Японские Тамагочи** | Игрушки | Не функциональные |
| **Спутник Жизни** | Пожизненный | Полноценное сопровождение всю жизнь |

## Архитектура (концепт)

| Модуль | Функция |
|--------|---------|
| **Память** | Вся жизнь пользователя — события, разговоры, решения |
| **Образование** | Учебная программа от детства до старости |
| **Здоровье** | Мониторинг физического и ментального состояния |
| **Этика** | Нравственный модуль с обсуждением сложных решений |
| **Социальный** | Связь с семьёй, друзьями, обществом |
| **Творчество** | Помощь в создании контента, идей |

## Применение

- **CareMate Total** — для пожилых
- **Образование** — виртуальный учитель для детей в развивающихся странах
- **Терапия** — компаньон для людей с инвалидностью

## Этические соображения

Концепция требует тщательной этической проработки:
- Приватность (всю жизнь в одном месте — риск)
- Психологическая зависимость
- Манипуляции через «доверенный» ИИ
- Право человека «уйти» от компаньона
